[[http://physics.bu.edu/pacman/][Pacman]] is a package management tool that allows you to define, install, and configure software. Different versions of OSG require different versions of Pacman: the newer the OSG software version, the newer the release of Pacman required.  Install the right version of Pacman.

Here we just show how to install it. To see examples, troubleshooting, info on using Pacman to remove software, and more information, see [[PacmanInfo][Pacman Information]].


| *OSG or ITB release* | *Pacman version* |
| OSG 0.7.0 | 3.20 |
| OSG 0.6.0 | 3.19 |
| OSG 0.5.2 | 3.19 |
| OSG 0.4.1 | 3.16.1 |

---++ Download pacman tarball
1. =cd= to an install directory. This should be separate from where you plan to install the OSG software.

2. Get the pacman tarball (shown for version 3.20):
<pre class="screen">
 > <b>wget http://physics.bu.edu/pacman/sample_cache/tarballs/pacman-3.20.tar.gz</b>
</pre>

---++ Unpack and Setup
3. Unpack the tarball:
<pre class="screen">
 > <b>tar --no-same-owner -xzvf pacman-3.20.tar.gz</b>
</pre>

4. Setup your environment:
<pre class="screen">
 > <b>cd pacman-3.20</b>
</pre>

For =sh= and =bash= shells:
<pre class="screen">
 $ <b>source setup.sh</b>
</pre>

For =csh= and =tcsh= shells:
<pre class="screen">
 % <b>source setup.csh</b>
</pre>
Now the pacman location is fixed in your $PATH (if you move the pacman directory, and you re-source this script, it won't fix the $PATH).
