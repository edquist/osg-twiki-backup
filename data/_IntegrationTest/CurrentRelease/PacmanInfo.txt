---++Installing and Setting Up Pacman
Pacman is a metapacking tool that allows you to define, install, and configure software.
The homepage of Pacman is http://physics.bu.edu/pacman/. 

Pacman Version 3.16.2 should be used for the ITB 0.3.6 / OSG 0.4.1 installations.
d31 1
---+++ Download
First, =cd= to an install directory somewhere.  Then:<br>
=> wget= http://physics.bu.edu/pacman/sample_cache/tarballs/pacman-latest.tar.gz
=> wget= http://physics.bu.edu/pacman/sample_cache/tarballs/pacman-3.16.1.tar.gz

---+++ Unpack and Setup
=> tar --no-same-owner -xzvf pacman-latest.tar.gz=
=> tar --no-same-owner -xzvf pacman-3.16.1.tar.gz=

=> cd pacman-3.16.2=
=> cd pacman-3.16.1=

For =sh= and =bash= shells:<br>
=$ source setup.sh=

For =csh= and =tcsh= shells:<br>
=% source setup.csh=

---+++ Installing OSG software using Pacman
=> mkdir INSTALLATION_DIRECTORY  (e.g, /usr/local/osg)= <br>
=> mkdir INSTALLATION_DIRECTORY  (e.g. /usr/local/osg)= <br>
=> pacman -get iVDGL:PACKAGE_NAME  (e.g ce-0.3.6)= 
=> pacman -get OSG:PACKAGE_NAME= for production 


If you get an error message indicating ".....is not yet supported", you may need to specify the pacmam command with the following argument:<br>
If you get an error message indicating ".....is not yet supported", you may need to specify the pacman command with the following argument:<br>
=> pacman -pretend-platform:RHEL-3 -get iVDGL:ce= <br>
where you could substitute any platform for RHEL-3.
To see a list of the supported pacman platforms, execute:<br>
To see a list of the defined pacman platforms (not all of which are yet supported), execute:<br>
=> pacman  -platforms= 


Pacman normally removes the <i>setup.sh</i> file  at the start of an installation and, if successful, restores the file.
You can suppress this removal feature by running with a  "-allow save-setup" option if you like or to recover a setup file you can  try running "pacman -setup" to recover a setup file. 
=> pacman -remove package_name= 
If you installed a package that you may later have deemed unnecessary, you can remove it  by running "pacman -remove package_name". 

For details on these commands see the <a href="http://physics.bu.edu/pacman">Pacman home page</a>.
