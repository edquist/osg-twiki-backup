---++Introduction

   Virtual Organizations can send registration information about their VirtualOrganizations/VOInfoMS service to the igoc at ivdgl dot org. The
   registration information that is needed is as follows:

   * *Unique VO* name
   * The URL which includes the hostname of the EDG_VOMS_ADMIN server and the group
      which should be queried.
   * Contact information for the system administrator in charge of the services.
   * The EDG-VirtualOrganizations/VOInfoMS service entry to be  distributed via the *vomses* file
      for voms-proxy-init client service. In the template format:
<verbatim>
"nickname" "host" "port" "hostcert's dn" "vo" "globus version (of the server to contact)"
</verbatim>

---++Sample Registration

<verbatim>

   Unique VO   iVDGL
   URL:        https://pki0.indiana.edu:8443/edg-voms-admin/g3dev/group1
   SysAdmin:   Leigh Grundhofer
   VOMSes:     "ivdgl" "pki0.indiana.edu" "15003" "/DC=org/DC=doegrids/OU=Services/CN=pki0.indiana.edu" "ivdgl" "32"          
</verbatim>
