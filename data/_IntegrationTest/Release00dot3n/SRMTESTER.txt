---++Introduction

SRM-TESTER is a validation tool for SRM based Storage Sites. It validates SRM spec v1.1 interface for compatibility and interoperability. It also verifies the operation of file transfer protocols such as gsiftp, http or ftp. The testing results may be published on a web page optionally.

---++Readiness Plan

SRM-TESTER is ready for the integration into the [[Trash/Trash/Integration.GridCat][GridCAT]]. The SRM-TESTER package is available for deployment through VDT or downloading from http://sdm.lbl.gov/srm-dist.
The sample published results can be viewed from the [[http://sdm.lbl.gov/cgi-bin/srm-tester/show-report.cgi][testing site]].

---++Documentation

Documentation is available from the web http://sdm.lbl.gov/srm-dist.

---++Supports

Supports are available through srm@lbl.gov.
