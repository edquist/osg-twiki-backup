Just before going through the section "Installing the pnfs Server", edit the file /opt/pnfs/tools/autoinstall-s.sh, changing the line

mount -o udp,intr,hard,rw,noac localhost:/fs ${pnfsMountDir}>>${logfile} 2>&1 \
to
mount -o udp,intr,hard,rw,noac,nfsvers=2 localhost:/fs ${pnfsMountDir}>>${logfile} 2>&1 \

Also edit the file /opt/pnfs/bin/pnfs, changing the line
mount -o udp,intr,rw,noac,hard localhost:/fs /pnfs/fs
to
mount -o udp,intr,rw,noac,hard,nfsvers=2 localhost:/fs /pnfs/fs

Then follow the pnfs Server instructions.
