---++Introduction
 
Updates found from Validation testing from ITB 0.3.6. 

---++Packages Affected
	* ce package which depends on srm-cp and VDT
	* wn-client which depends on srm-cp
	* srm-cp package version updated to 1.23 to correct an error and allow non-root installation
	* site-verify package clean up of left installation directory
	* client package updated to include the Globus-WS-Client package
	* osg-auto-0.3.7 - osg-version updated to reflect the new version
	* VDT 1.3.10b - full details [[http://vdt.cs.wisc.edu/releases/1.3.10/release.html][here]]
		* Condor  updated to 6.7.18
		* gridftp fix for permission problems
		* globus-url-copy creation of directories
		* created the gridftp.log file to prevent problems with VO_IO modules
