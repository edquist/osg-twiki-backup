---++ Introduction
The purpose of this topic is to establish best practices for creating the documentation as we go through the integration process.

---++ Release version scheme
We will refer to an ITB cycle by its series name, ITB 0.5.n, where its understood that the result of this will process will be a release candiate for OSG 0.6.  Each minor version (ITB 0.5.0, 0.5.1, ...) will share the same documentation set but differences between the minor versions will be separately described in-line.
