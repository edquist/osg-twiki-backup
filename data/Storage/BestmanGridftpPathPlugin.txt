---++ Introduction
This page documents a  way for Bestman-Gateway (or Bestman2)
to select the best gridftp server to use depending on the storage path 
of the file to be accessed, depening on which server has the storage path
locally mounted.

---++ Checklist
This document assumes a working Bestman-Gateway installation on top of 
a posix compliant file system, which has Bestman-Gateway installed in ==/opt/bestman/==,
or Bestman2-Gateway in ==/opt/bestman2/==. Otherwise you will have to adjust the 
configuration files accordingly.

You will need two files for this configuration: ==MountPoint.java== and ==servers.txt==;
examples of both are attached to this page.

---++ Installation and Configuration
Edit ==MountPoint.java==: change ==log_filename==, ==server_filename==, and ==defaultServer==
according to your local setup.

Customize ==servers.txt== for your site and file system structure.

Then issue the following commands:

<pre class="screen">
cp -p servers.txt MountPoint.java /opt/bestman/bestman/lib/plugin
cd /opt/bestman/bestman/lib/plugin
javac -classpath "/opt/bestman/bestman/lib/Berkeley.StorageResourceManager.jar" MountPoint.java
jar cvf mount.jar MountPoint.class
cp -p /opt/bestman/bestman/conf/bestman.rc /opt/bestman/bestman/conf/bestman.rc.orig
</pre>

Add the following to ==/opt/bestman/bestman/conf/bestman.rc==:

<pre class="file">
###########################################################
# gridftp server selection plugin 
###########################################################
protocolSelectionPolicy=class=MountPoint&jarFile=mount.jar&name=gsiftp
pluginLib=/opt/bestman/bestman/lib/plugin
</pre>

<pre class="screen">
cp -p /opt/bestman/bestman/conf/bestman.rc 
/opt/bestman/bestman/conf/bestman.rc.save

/etc/init.d/bestman stop
/etc/init.d/bestman start
</pre>

Copying the modified ==bestman.rc== to ==bestman.rc.save== is recommended
since this modification may be undone by re-running ==configure_bestman==.

For Bestman2, replace all instances of ==bestman== with ==bestman2==
(in files and directory paths).
