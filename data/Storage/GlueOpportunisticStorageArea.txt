---++++++ Glue Schema 1.3 Elements for Opportunistic Storage

Here is an example glue schema 1.3 SA element for an opportunistic storage area.

<verbatim>
#
# Replica-Online SA for opportunistic use
#
dn: GlueSALocalID=opportunistic:replica:online,GlueSEUniqueID=oursite.edu,mds-vo-name=OURSITE,o=grid
objectClass: GlueSATop
objectClass: GlueSA
objectClass: GlueSAPolicy
objectClass: GlueSAState
objectClass: GlueSAAccessControlBase
objectClass: GlueKey
objectClass: GlueSchemaVersion
GlueSAPath: /pnfs/oursite.edu/data/public
GlueSALocalID: opportunistic:replica:online
GlueSAStateAvailableSpace: 6000000000000
GlueSAStateUsedSpace: 3000000000000
GlueSAAccessControlBaseRule: VO:cdf
GlueSAAccessControlBaseRule: VO:dzero
GlueSAAccessControlBaseRule: VO:gadu
GlueSAAccessControlBaseRule: VO:sdss
GlueSAAccessControlBaseRule: VO:ligo
GlueSAAccessControlBaseRule: VO:star
GlueSAAccessControlBaseRule: VO:ivdgl
GlueSAAccessControlBaseRule: VO:gridex
GlueSAAccessControlBaseRule: VO:fmri
GlueSAAccessControlBaseRule: VO:mis
GlueSAAccessControlBaseRule: VO:grase
GlueSAAccessControlBaseRule: VO:glow
GlueSARetentionPolicy: replica
GlueSAAccessLatency: online
GlueSAExpirationMode: releaseWhenExpired
GlueSATotalOnlineSize: 1000000
GlueSAUsedOnlineSize: 800000
GlueSAFreeOnlineSize: 200000
GlueSAReservedOnlineSize: 0
GlueSATotalNearlineSize: 9000000
GlueSAUsedNearlineSize: 3000000
GlueSAFreeNearlineSize: 6000000
GlueSAReservedNearlineSize: 0
GlueChunkKey: GlueSEUniqueID=oursite.edu
GlueSchemaVersionMajor: 1
GlueSchemaVersionMinor: 3
</verbatim>

If you would like to publish a space token description for opportunistic use, add a <nop>VOInfo element.

<verbatim>
#
# space token description for opportunistic use
#
dn: GlueVOInfoLocalID=opportunistic-voinfo,GlueSALocalID=opportunistic:replica:online,GlueSEUniqueID=oursite.edu,mds-vo-name=OURSITE,o=grid
objectClass: GlueSATop
objectClass: GlueVOInfo
objectClass: GlueKey
objectClass: GlueSchemaVersion
GlueVOInfoLocalID: opportunistic-voinfo
GlueVOInfoName: opportunistic-storage
GlueVOInfoPath: /pnfs/oursite.edu/data/public
GlueVOInfoTag: public-token
GlueVOInfoAccessControlBaseRule: VOMS:/cdf
GlueVOInfoAccessControlBaseRule: VOMS:/dzero
GlueVOInfoAccessControlBaseRule: VOMS:/gadu
GlueVOInfoAccessControlBaseRule: VOMS:/sdss
GlueVOInfoAccessControlBaseRule: VOMS:/ligo
GlueVOInfoAccessControlBaseRule: VOMS:/star
GlueVOInfoAccessControlBaseRule: VOMS:/ivdgl
GlueVOInfoAccessControlBaseRule: VOMS:/gridex
GlueVOInfoAccessControlBaseRule: VOMS:/fmri
GlueVOInfoAccessControlBaseRule: VOMS:/mis
GlueVOInfoAccessControlBaseRule: VOMS:/grase

GlueChunkKey: GlueSALocalID=opportunistic:replica:online

GlueSchemaVersionMajor: 1
GlueSchemaVersionMinor: 3
</verbatim>
