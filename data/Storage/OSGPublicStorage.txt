---++ Allocation Accounting Service

Accounts for allocations.

---+++ Storage area discovery utility

Queries BDII and displays space and authorization info for storage areas.

*Script Name*

get_storage_area


*Usage*

<pre>
get_storage_area [--vo <VOname1[,VOname2]...>] [--storage_element_id <GlueSEUniqueID>] [--free_online_size <GlueSAFreeOnlineSize>] [--show_site_name] [--hide_vo]
</pre>

*Parameters*
%TABLE{ headerbg="#eeeeee" headercolor="#000000" databg="#ffffff" tableborder="1" columnwidths="120," cellpadding="2" cellspacing="1" dataalign="left" valign="top" sort="off"}%
| *name* | *definition* | *required* |
|--vo|Restrict to storage areas for which all of the named Virtual Organizations are authorized.|no|
|--match_any|Restrict to storage areas for which any of the named VOs is authorized.|no|
|--storage_element_id|Restrict to a specific Storage Element. Glue schema attribute is !GlueSEUniqueID.|no|
|--free_online_size|Restrict to SAs with minimum available space, in MB. Glue schema attribute is !GlueSAFreeOnlineSize.|no|
|--show_site_name|Show the site name for each storage area. Default is to show the Storage Element ID.|no|
|--hide_vo|Do not show the VO name in the output.|no|

*Example Output*
<verbatim>
[aas]$ get_storage_area
STORAGE ELEMENT ID                      STORAGE AREA NAME                                           TOTAL     FREE      VOs
atlas.bu.edu                            Default Storage Area                                        122984    17059     atlas,cdf,mis,ops,osg
atlas11.hep.anl.gov                     Default Storage Area                                        0         0         atlas
atlfs02.phy.duke.edu                    Default Storage Area                                        0         0         atlas
bsrm-1.t2.ucsd.edu                      Default Storage Area                                        1029054   164169    accelerator,astro,atlas,cdf,... 
charm.ucr.edu                           Default Storage Area                                        6730      1236      atlas,cdf,cms,engage,fermilab,mis,nebiogrid,nysgrid,sbgrid,star
...
</verbatim>
