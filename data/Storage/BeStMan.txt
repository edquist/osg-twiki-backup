---+++ Berkeley Storage Manager (!BeStMan) SRM v2.2 status

!BeStMan is a full implementation of SRM v2.2 for disks and mass storage systems such as HPSS. It works on top of existing file system, and has been reported so far to work on file systems such as NFS, PVFS, AFS, GFS, and Lustre. It also works with any existing file transfer service.
