---+ Making RSV work with HDFS
Because RSV uses a ":" character in its file names, and HDFS does not support this character (as it is illegal in Windows), RSV currently must be patched in order to work with HDFS.

Start off with RSV v2.4.1 (released in the ITB, but not yet in Production; should be coming within a week or so of this writing).  Apply the below patch.
