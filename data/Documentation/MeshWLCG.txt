We are using the so-called "mesh-configuration" to manage the WLCG perfSONAR instances.  The perfSONAR developers have documented the details at http://docs.perfsonar.net/multi_overview.html

OSG is working on a mesh creation and management system but it is not quite ready for production.   In the interim we are providing all the existing 16 meshes in use for WLCG via OSG. 

   * CA mesh http://myosg.grid.iu.edu/pfmesh/json/name/ca-all
   * DE mesh http://myosg.grid.iu.edu/pfmesh/json/name/de-all
   * ES mesh http://myosg.grid.iu.edu/pfmesh/json/name/es-all
   * FR mesh http://myosg.grid.iu.edu/pfmesh/json/name/fr-all
   * ITATLAS mesh http://myosg.grid.iu.edu/pfmesh/json/name/it-atlas
   * ITCMS mesh http://myosg.grid.iu.edu/pfmesh/json/name/it-cms
   * LHCONE mesh http://myosg.grid.iu.edu/pfmesh/json/name/lhcone-all
   * ND mesh http://myosg.grid.iu.edu/pfmesh/json/name/nd-all
   * LHCOPN mesh http://myosg.grid.iu.edu/pfmesh/json/name/opn-all
   * RU mesh http://myosg.grid.iu.edu/pfmesh/json/name/ru-all
   * TW mesh http://myosg.grid.iu.edu/pfmesh/json/name/tw-all
   * UK mesh http://myosg.grid.iu.edu/pfmesh/json/name/uk-all
   * USATLAS mesh http://myosg.grid.iu.edu/pfmesh/json/name/us-atlas
   * USCMS mesh http://myosg.grid.iu.edu/pfmesh/json/name/us-cms
   * WLCG mesh http://myosg.grid.iu.edu/pfmesh/json/name/wlcg-all

Most sites should configure two meshes:  their regional cloud/VO mesh and the WLCG mesh.   Some sites may also be participating in the LHCONE or LHCOPN meshes and those sites should add those meshes.  You can check if your host is in a mesh by opening one of the URLs above and search for your host DNS name.
