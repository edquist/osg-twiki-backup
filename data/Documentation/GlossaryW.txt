   $ WN: _See_ [[#DefsWorkerNode][Worker Node]].
   $ Wall Hours<a name=DefsWallHours"></a>: The number of hours taken by a computer to complete a task, including [[#DefsWorkerNode][CPU hours]] + I/O time + any communication channel delay

   $ Worker Node<a name="DefsWorkerNode"></a>: The terminology used in the OSG to name the node that is executing jobs on the behalf of grid users. Each worker node is only associated with one %LINK_GLOSSARY_CE%. A Compute Element can be associated with more than one Worker Node.
