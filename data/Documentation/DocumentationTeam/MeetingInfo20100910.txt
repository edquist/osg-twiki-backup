---++ Tips on Reviewing the Structure of Documents

   * Before the review take note about what goals you think the page will serve taking 'About this Document' into account.
   * Evaluate the table of contents and check if you can clearly identify the goals of the page.
   * Evaluate the table of contents and check if high level goals are structured above low level goals and not the other way around.
   * Run a few usage examples. Make sure you can quickly navigate to high level goals (renew grid cert, etc)
   * Check if the page provides troubleshooting information

---++ About INCLUDE

   * see [[http://twiki.org/cgi-bin/view/TWiki/VarINCLUDE][manual page]]
   * define the SHIFT_TOC variable on page that defines the content to be included

<pre class="file">
<verbatim>
<!-- define zero shift for current document
   * Local SHIFT_TOC = 
-->
</verbatim>
</pre>

   * change header definitions on page that contains content to be included elsewhere

<pre class="file">
<verbatim>
---+ Level 1 Header
to
---%SHIFT_TOC%+ Level 1 Header
</verbatim>
</pre>

   * create INCLUDE Sections on page that contains content to be included elsewhere
 
<pre class="file">
<verbatim>
%STARSECTION{"SomeName"}%
...
%ENDSECTION{"SomeName"}%
</verbatim>
</pre>

   * use INCLUDE statement on arbitrary pages

<pre class="file">
<verbatim>
%INCLUDE{"ReleaseDocumentation/StartingServices" section="About" SHIFT_TOC="++"}%
</verbatim>
</pre>

   * see example [[ReleaseDocumentation/StartingServices]]
