---++ About INCLUDE (updated)

   * combination of Marco's and Robert's earlier solution
   * see [[http://twiki.org/cgi-bin/view/TWiki/VarINCLUDE][manual page]]

   * change header definitions on page that contains content to be included elsewhere

<pre class="file">
<verbatim>
---+ Level 1 Header
to
---%SHIFT%+ Level 1 Header
</verbatim>
</pre>

   * create INCLUDE Sections on page that contains content to be included elsewhere
 
<pre class="file">
<verbatim>
%STARSECTION{"SomeName"}%
...
%ENDSECTION{"SomeName"}%
</verbatim>
</pre>

   * use INCLUDE statement on arbitrary pages

<pre class="file">
<verbatim>
%INCLUDE{"ReleaseDocumentation/StartingServices" section="About" TOC_SHIFT="++"}%
</verbatim>
</pre>

   * see example [[ReleaseDocumentation/StartingServices]]
