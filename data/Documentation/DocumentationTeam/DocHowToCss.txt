---++ *Integrating Cascading Style Sheets into Twiki*

Checkout a working copy of the [[DocRepositoryCss][Content Management Repository]] using the =username= and =password= that was provided to you by Main.RobertEngel.

<div class="screen-top">
<pre class="screen-content">
[robert@osg-job ~]$ svn co --username=your.username svn://osg-vtb.ligo.caltech.edu/home/subversion/repos/OSG-CM/trunk OSG-CM
A    OSG-CM/twiki
Checked out revision 1185.
</pre>
</div>

Skip the %RED%previous%ENDCOLOR% step if you already checked out a working copy. In this case it is only necessary to update the working copy:

<div class="screen-top">
<pre class="screen-content">
[robert@osg-job ~]$ cd OSG-CM
[robert@osg-job OSG-CM]$ svn update
At revision 1185.
</pre>
</div>

Make changes to .css files located in OSG-CM/twiki/css and commit changes to the repository after you are done:

<div class="screen-top">
<pre class="screen-content">
[robert@osg-job ~]$ cd OSG-CM
[robert@osg-job OSG-CM]$ svn commit
</pre>
</div>

The web server will sync the Content Management Repository every minute and make the content available as http://osg-job.ligo.caltech.edu/OSG-CM/trunk/twiki/css/ . 

To integrate a style sheet into the source code of the Twiki page you wish to style, just include its %BLUE%URL%ENDCOLOR% inside a %RED%HTML comment%ENDCOLOR% at the end of the page:

<div class="screen-top">
<verbatim>
<!--
   * Set USERSTYLEURL = http://osg-job.ligo.caltech.edu/OSG-CM/trunk/twiki/css/sample.css
-->
</verbatim>
</div>
