   $ BDII<a name="DefsBDII"></a>: (Berkeley Database Information Index) is an LCG implementation of Globus GIIS-like information index based on the Berkeley Database.

   $ !BeStMan: Berkeley Storage Manager (!BeStMan) is an LBNL implementation of SRM based on Unix disks and HPSS. 

   $ brokering<a name="DefsBrokering"></a>: (in SRMs) when a file is not locally available, contacting the source location to [[#DefsPp][pin]] a file and then invoke a transfer service to get the file.

   $ bit file ID<a name="DefsBitFileId"></a>: An Enstore-assigned, unique identifier for a data file.
