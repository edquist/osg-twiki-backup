   $ JClarens: JClarens is a Java-based supplement to the Python-based Clarens Web Services Framework developed at Caltech.

   $ Job: An executable set of code; in OSG, an executable to be submitted to run on OSG resources.

   $ Job manager: A Globus term that refers to a program used to manage jobs at a grid site (e.g., LSF, PBS, Condor).
