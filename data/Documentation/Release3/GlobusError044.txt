---++Globus Error: 44

---+++Syntax:

=GRAM Job failed because the job manager failed to stage the stdin file (error code 44)=

---+++Known causes:

This occurs when the server does not have permission to read the file defined as stdin or that file does not exist.

---+++Recommendations:

Make sure the file exists.

Check permissions on the file defined as stdin and the directory in which it is present.
