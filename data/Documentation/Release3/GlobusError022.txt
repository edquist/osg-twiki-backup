---++Globus Error: 22

---+++Syntax:

=GRAM Job submission failed because the job manager failed to create an internal script argument file (error code 22)=

---+++Known causes:

This is known to occur when the user does not have a home directory on the gatekeeper .

---+++Recommendations:

Contact site administrator and make sure that home directory is defined.
