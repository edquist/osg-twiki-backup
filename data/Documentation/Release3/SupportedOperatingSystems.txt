---+ OSG Software 3.0 Supported Operating Systems
%DOC_STATUS_TABLE%

OSG Software 3.0 is supported on Red Hat Enterprise Linux 5 and 6 and clones. We support both 32-bit and 64-bit variants. Specifically:

| *Operating System* | *Comment* |
| Red Hat Enterprise Linux 5 | |
| Scientific Linux 5 | |
| !CentOS 5 | |
| Red Hat Enterprise Linux 6 | Only supports worker-node and glexec |
| Scientific Linux 6 | Only supports worker-node and glexec |
| !CentOS 6 | Only supports worker-node and glexec |

Note that on RHEL 6 we currently only support the worker node and glexec. Full support is underway, and expected for April 2012.
