---++Globus Error: 135

---+++Syntax:

<verbatim>
GRAM Job failed because the job manager could not stage in a file (error code 135)
</verbatim>

---+++Known causes:

This error usually occurs when the server cannot read the input file from local host.

---+++Recommendations:

Check permissions on the directory from which the input is supposed to be read and make sure that it is readable.
