---++Globus Error: 136

---+++Syntax:

=Globus error 136: the scratch directory could not be created=

---+++Known causes:

Globus GRAM requires a scratch directory and it does not have read/write permissions for the directory it is attempting to use as scratch.

---+++Recommendations:

Ask the site admin to check scratchDirectory and be certain permissions are set correctly.
