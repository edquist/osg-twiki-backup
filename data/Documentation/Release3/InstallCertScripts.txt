---++ Install Cert Scripts - %RED%Early Draft%ENDCOLOR%

---++ Introduction

The OSG Cert Scripts are used for requesting and retrieving host and service certificates. 

---++ Installation Procedure for OSG Cert Scripts

The Cert Scripts package can be installed with the following command:
<pre class="rootscreen">
%UCL_PROMPT_ROOT% yum install osg-cert-scripts</pre>

---++ Requesting a Host Cert
Ran into issues with running <pre>cert-request</pre> due to:

<pre>Crypt::SSLeay not installed</pre>

Installed with:

<pre class="rootscreen">yum install perl-Crypt-SSLeay</pre>

---++ Retrieving a Host Cert

You will receive an email from DOEGrids containing a command that looks like this:

<pre class="rootscreen">cert-retrieve -serialnum %RED%11111%ENDCOLOR%</pre>
