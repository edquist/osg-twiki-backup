---++Condor Hold Code 7

---+++Syntax

=Code 7 subcode &lt;unix_errno&gt;=

---+++Known causes:

Hold Code 7 indicates that condor could not open the standard output file.

---+++Recommendations:

Check your firewall settings and see [[Trash/TroubleshootingCondorFirewalls][Condor ports]].

Check file/folder permissions to be sure that Condor can write the output file.
