---++Introduction

SRM-TESTER is a validation tool for SRM based Storage Sites. It validates SRM spec v1.1 and v2.2 interface for compatibility and interoperability. It also verifies the operation of file transfer protocols such as gsiftp, http or ftp. The testing results may be resulted in a file so that web publishing is possible.

---++Availability

The SRM-TESTER package is available for deployment through VDT or downloading from [[http://datagrid.lbl.gov/bestman/#tester][http://datagrid.lbl.gov/bestman/#tester]].  OSG SRM based storage is being tested daily, and the results are published on [[http://datagrid.lbl.gov/osg][http://datagrid.lbl.gov/osg]].  Any new OSG storage sites are recommended to register at [[http://datagrid.lbl.gov/sitereg][http://datagrid.lbl.gov/sitereg]].


---++Documentation

Documentation is available from the web, chapter 2 from [[http://datagrid.lbl.gov/bestman/bestman-manual.html][here]].
