---++Condor Hold Code 8

---+++Syntax

=Code 8 subcode &lt;unix_errno&gt;=

---+++Known causes:

Hold Code 8 indicates that condor could not open the standard input file.

---+++Recommendations:

Check your firewall settings and see [[CondorFirewalls][Condor ports]].

Check file permissions and make sure that condor can read the input file.
