---++ Release Variables

%EDITTHIS%

A set of standard variables are provided so that items that may change with each release of software are automatically updated in the documentation.  You should use the variables any time you want to refer to the corresponding item rather than the verbatim numbers or text.  The variables are set in the Documentation.WebPreferences TWiki page.
   * OSG release variable: *%<nop>OSG_VERSION%* = %OSG_VERSION%
   * OSG grid: *%<nop>OSG_GRID%* = %OSG_VERSION%
   * Supported OS: *%<nop>SUPPORTED_OS%* = %SUPPORTED_OS%
   * ITB Version: *%<nop>ITB_VERSION%* = %ITB_VERSION%
   * Release Status: *%<nop>RELSTATUS%* = %RELSTATUS%
   * OSG Release: *%<nop>OSG_RELEASE%* = %OSG_RELEASE%

%STOPINCLUDE%
