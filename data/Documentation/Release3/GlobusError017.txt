---++Globus Error: 17

---+++Syntax:

=GRAM Job failed because the job failed when the job manager attempted to run it (error code 17)=

---+++Known causes:

This is known to occur when a user is submitting a job using Condor and the Condor configuration (condor_config) file cannot be located.

---+++Recommendations:

Set $CONDOR_CONF to the location of your condor_config file by exporting it in an initialization script.
