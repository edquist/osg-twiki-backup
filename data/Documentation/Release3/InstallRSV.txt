First, install condor-cron.  This will be done automatically when installing RSV in the future, but it doesn't yet work.
<pre class="screen">
$ yum install condor-cron
$ /etc/init.d/condor-cron start
</pre>

Then install RSV and turn on a few probes for your HOSTNAME.
<pre class="screen">
$ yum install rsv
$ rsv-control --enable org.osg.general.ping-host org.osg.general.vo-supported org.osg.general.osg-version org.osg.globus.gram-authentication org.osg.globus.gridftp-simple org.osg.general.osg-directories-CE-permissions -u <em>HOSTNAME</em>
$ /etc/init.d/rsv start
</pre>

Then you can run the metrics by hand, or examine the log files after they have run via condor-cron.  The HTML and Gratia consumers are not currently operational.
<pre class="screen">
$ rsv-control --run --all-enabled
</pre>


There may be some odd permissions problems between the 'rsv' and 'cndrcron' users.  I noticed some in testing but I think it may be due to a corrupted development/testing environment.
