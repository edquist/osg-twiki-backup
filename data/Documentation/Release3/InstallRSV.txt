---++ Install Instructions
%INCLUDE{"InstallVDTRepo" headingoffset="1"}%
%INCLUDE{"InstallCACerts" headingoffset="1"}%

To install RSV:
   * Install your service certificate at =/etc/grid-security/rsvcert.pem= and the service key at =/etc/grid-security/rsvkey.pem=

   * Install RSV via RPMs
<pre class="screen">
%UCL_PROMPT_ROOT% yum  --enablerepo=osg-testing install rsv
</pre>

   * Edit =/etc/osg/config.d/30-rsv.ini=

   * Configure and start RSV
<pre class="screen">
%UCL_PROMPT_ROOT% osg-configure -v
%UCL_PROMPT_ROOT% osg-configure -c
%UCL_PROMPT_ROOT% /etc/init.d/rsv start
</pre>

Then you can run the metrics by hand, or examine the log files after they have run via condor-cron.  The HTML and Gratia consumers are not currently operational.
<pre class="screen">
$ rsv-control --run --all-enabled
</pre>
