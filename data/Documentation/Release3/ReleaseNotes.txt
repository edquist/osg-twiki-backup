---++ What's New in Release 3
Release 3 introduces use of RPMs for installation of most of the software packages in OSG.
This is a major change in methodology that affects nearly every aspect of installing the release.
This document outlines some requirements for moving to the release and presents the changes you need to 
know to successfully install, configure, and administer the release.

NOTE to Owner:  Check out [[https://twiki.grid.iu.edu/bin/view/Documentation/Release3/RPMWhatsNew]]
---++ Requirements

   * *OS:* Scientific Linux 5, !CentOS 5, or Red Hat Enterprise Linux 5
   * Make sure you don't have Globus pre-installed from EPEL.
