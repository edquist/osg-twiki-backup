---+!! Installing OSG CA certificates into a web browser

%DOC_STATUS_TABLE%
%TOC%

---+ About This Document

This document contains instructions on how to install OSG Certifying Authority certificates into a web browser.  This enables the web browser to verify that connections to web servers that use OSG host certificates are secure.

%STARTSECTION{"InstallCACerts"}%

---+ Downloading Certifying Authority certificate files for OSG !CILogon-based certificates
[[http://ca.cilogon.org/downloads][Here]] you will find the location from which you can download the CILogon OSG CA root cert that is needed to be added to the OS X Keychain (for example) or to the trusted issuer stores in the various browsers.

The following is a sample of the page.  

       %ATTACHURL%/OSG_Repository_root_SHA-2_CA.png  %BR%

---++ Installing Certifying Authority certificates into a web browser

To import the CA certificates into your browser, download the [[https://cilogon.org/cilogon-osg.pem][Root]] PEM format file and then follow these instructions for each of them.  These instructions are for Firefox but most browsers have something similar.

   1. Go to Preferences -> Advanced -> Certificates -> View Certificates -> Authorities:
       <p> %ATTACHURL%/CA_Notpresent.png</p>
   1. Click "Import" and select the certificate just downloaded. 
      <p> %ATTACHURL%/CA_Add.png  </p>
   1. After you select the certificate, this window will be shown
      <p> %ATTACHURL%/Certificate_Added.png  </p>
      Click on the first two "Trust" boxes and click "OK" to save it.
    1. Successful import will show the new CA as follows
      <p> %ATTACHURL%/CA_present.png  </p>

%ENDSECTION{"InstallCACerts"}%
