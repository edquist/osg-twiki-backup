---+!! Installing OSG CA certificates into a web browser

%DOC_STATUS_TABLE%
%TOC%

---+ About This Document

This document contains instructions on how to install OSG Certifying Authority certificates into a web browser.  This enables the web browser to verify that connections to web servers that use OSG host certificates are secure.

%STARTSECTION{"InstallCACerts"}%

---+ Downloading Certifying Authority certificate files for OSG !DigiCert-based certificates
[[https://www.digicert-grid.com/][Here]] you will find the location from which you can download the Digicert root and intermediate CA certs that are needed to be added to the OS X Keychain (for example) or to the trusted issuer stores in the various browsers.

The following is a sample of the page.  The links circled in red are the files that are needed to be downloaded and imported.

       %ATTACHURL%/DigiCert_Repository_root_SHA-2_CA.png  %BR%

---++ Installing Certifying Authority certificates into a web browser

To import the CA certificates into your browser, download the [[https://www.digicert-grid.com/DigiCertGridRootCA.cer][Root]], [[https://www.digicert-grid.com/DigiCertGridCA-1.cer][CA-1]] and [[https://www.digicert-grid.com/DigiCertGridCA-1G2-Classic-2015.cer][SHA-2 CA-1]] PEM format files and then follow these instructions for each of them.  These instructions are for Firefox but most browsers have something similar.

   1. Go to Preferences -> Advanced -> Certificates -> View Certificates -> Authorities:
       <p> %ATTACHURL%/Certificates_Manager.png</p>
   1. Click "Import" and select the certificate just downloaded. 
      <p> %ATTACHURL%/Certificate_Import.png  </p>
   1. After you select the certificate, this window will be shown
      <p> %ATTACHURL%/Certificate_Added.png  </p>
      Click on the first two "Trust" boxes and click "OK" to save it.

%ENDSECTION{"InstallCACerts"}%
