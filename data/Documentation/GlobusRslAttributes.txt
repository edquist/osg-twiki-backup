---+ Globus RLS Attributes for PBS
This stub describes the attributes for PBS listed in the Globus <span class="firstterm">Resource Specification Language</span> (RSL).

%STARTINCLUDE%
%EDITTHIS%

---++++ Condor
|  *Attribute*  |  *Description*  | 
| condorsubmit | Allow the client to specify arbitrary additional attributes to be included in the Condor submit description file. |

---++++ PBS

|  *Attribute*  |  *Description*  |  *Values*  |
| =email_address= | Set the email address to receive notifications. See the email_on_abort, email_on_execution, and emailontermination attributes. |  | 
| =email_on_abort= | Send email to the job submitter (or the address specified in the email_address RSL attribute if present) if the job is aborted by the scheduler. | yes no |
| =email_on_execution= | Send email to the job submitter (or the address specified in the email_address RSL attribute if present) when the job begins execution. | yes no |
| =email_on_termination= | Send email to the job submitter (or the address specified in the email_address RSL attribute if present) when the job terminates. | yes no |

%STOPINCLUDE%
