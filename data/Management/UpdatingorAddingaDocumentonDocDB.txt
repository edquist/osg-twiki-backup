---++Adding a New Document to [[https://osg-docdb.opensciencegrid.org][DocDB]]
Use this procedure only when adding a completely new document to DocDB that has an attachment, for any updates or documents referring to links please see below.

   * Open [[http://osg-docdb.opensciencegrid.org][DocDB]]
   * If you are unsure of where to place the document you can instead click on [[https://osg-docdb.opensciencegrid.org/topics_cert.html][Guidelines on Which Topics to Use]]
   * Select *Certificate Version of OSG DocDB*
   * Select *Create or change documents or other information*
   * Scroll down to the orange *Customized Insert/Modify* box
   * Select *New Document*
   * Submit all relevant information. 
   * Be sure to enter the document in the correct topic (revisit the Topic Guidelines if you are unsure) and the correct keyword if it is applicable.

---++Updating a Document in [[https://osg-docdb.opensciencegrid.org][DocDB]]

   * Open [[http://osg-docdb.opensciencegrid.org][DocDB]]
   * If you are unsure of where to place the document you can instead click on [[https://osg-docdb.opensciencegrid.org/topics_cert.html][Guidelines on Which Topics to Use]]
   * Select *Certificate Version of OSG DocDB*
   * Select *Create or change documents or other information*
   * Scroll down to the orange *Customized Insert/Modify* box
   * Select *Update DB information* if just changing location or other document database information
   * Select *Update Existing Document* if uploading a new version of a document
   * When selecting Update DB information be sure to select override submit date in the bottom of the box, and re-enter the original date.  

---++Creating a document with a link to an exterior page

   * Open [[http://osg-docdb.opensciencegrid.org][DocDB]]
   * If you are unsure of where to place the document you can instead click on [[https://osg-docdb.opensciencegrid.org/topics_cert.html][Guidelines on Which Topics to Use]]
   * Select *Certificate Version of OSG DocDB*
   * Select *Create or change documents or other information*
   * Scroll down to the orange *Customized Insert/Modify* box 
   * Select *Reserve DB #*
   * Complete form, with link to exterior located in the abstract
