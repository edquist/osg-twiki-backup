---+Atlas Software Installation on TeraGrid
 
Atlas software package has mainly three parts to be installed on a grid site:

1. Capone (usatlas production executor) server side package, i.e. gce-server, which defines the high level transformation scripts, executables and other tools used to invoke the grid jobs. 

2. JobTransformations, which has the low level scripts for running athena jobs with the appropriate jobOptions

3. Multiple Atlas releases, each one corresponds to an individual Pacman subdirectory. Each release takes about 4GB disk space.

The installation is based on pacman.

At this moment it's installed on the following sites:

---++UC

 Login hostname is tg-viz-login.uc.teragrid.org. 

 Atlas software is installed under /soft/community/osg/atlas_app

1) Get Pacman 

> cd /soft/community/osg/
> wget http://physics.bu.edu/pacman/sample_cache/tarballs/pacman-latest.tar.gz
> tar xvfz pacman-latest.tar.gz
> cd pacman-3.12.1; source setup.sh

2) Install capone server package

> mkdir atlas_app
> cd atlas_app
> pacman -get GCL:ACE-Server-dev -trust-all-caches -allow tar-overwrite

3) Install atlas software release 10.0.1

> mkdir -p atlas_app/atlas_rel/10.0.1
> cd atlas_app/atlas_rel/10.0.1
> pacman -get GCL:10.0.1slc3+gcc  -trust-all-caches -allow tar-overwrite
Note: You want to install the slc3 version because this UC cluster runs SuSE Linux Enterprise 
Server 8.

4) Install atlas software jobtransformations

> mkdir -p atlas_app/atlas_rel/kitval
> cd atlas_app/atlas_rel/kitval
> pacman -get GCL:JobTransforms -trust-all-caches -allow tar-overwrite
