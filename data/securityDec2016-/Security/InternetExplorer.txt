---+ Internet Explorer
---++ IE version 6
---+++ View Certificates
Click %ORANGE% Tools / Internet Options ... / Content Tab / Certificates ... %ENDCOLOR% button.

     <img src="%ATTACHURLPATH%/view-certificates.jpg" alt="view-certificates.jpg" width='509' height='471' />

The set of certificates that belong to you (for which the private key is stored in your browser)
are shown under the %ORANGE% Personal %ENDCOLOR% tab.  If you import your certificate but don't
have the private key in your browser then it will show up under the %ORANGE% Other People %ENDCOLOR% tab.

---+++ Edit CA trust settings
To see the Root CA certificates in your browser, 
click %ORANGE% Tools / Internet Options ... / Content Tab / Certificates ... %ENDCOLOR% button.  Then
click on the %ORANGE% Trusted Root Certificate Authorities %ENDCOLOR% tab.  If you have the DOEGrids CA chain
installed inyour browser then you should see the ESnet Root CA as shown below.

     <img src="%ATTACHURLPATH%/view-ca-certs.jpg" alt="view-ca-certs.jpg" width='509' height='471' />

 Click on the %ORANGE% View %ENDCOLOR% button to show more detailed information, as below.

     <img src="%ATTACHURLPATH%/esnet-root-view.jpg" alt="esnet-root-view.jpg" width='409' height='483' />

If it shows the intended purposes as all issuance policies and all application policies like the that above then
you don't need to change any settings.  However, if it indicates that the CA can not be used for some
application purposes then you should click the %ORANGE% Details %ENDCOLOR% tab

     <img src="%ATTACHURLPATH%/esnet-root-details.jpg" alt="esnet-root-details.jpg" width='409' height='483' />

and the %ORANGE% Edit Properties %ENDCOLOR% button, and you will get a window like the one below.

     <img src="%ATTACHURLPATH%/root-ca-edit-properties.jpg" alt="root-ca-edit-properties.jpg" width='409' height='483' />

Select the %ORANGE% Enable all purposes for this certificate %ENDCOLOR% radio button and then click all the OK and Close buttons to get back to your web browsing.
