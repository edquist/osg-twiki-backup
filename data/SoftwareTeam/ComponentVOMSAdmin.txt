---+ OSG Software Component: VOMS Admin

---++ Purpose

[[https://wiki.italiangrid.it/twiki/bin/view/VOMS/VOMSAdminUserGuide VOMS Admin]] provides a web interface to the [[ComponentVOMS][VOMS]] service, so that people can maintain the membership data contained therein. VOMS Admin is a web application that runs in Tomcat.
