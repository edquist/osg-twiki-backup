---# Testing Instructions

---## In the gums server

   1 Add an account mapper called test, type manual, persistence factory: mysql
   1 Add a user group called test, type manual, GUMS Acces: Read Self
   1 Add a group to account mapping called test, user group: test, account mapper test
   1 Add a host to group mapping: Hosts: */?*.fnal.gov, Group To Account Mapping: test
   1 Install privilege-xacml <pre class="rootscreen">
%UCL_PROMPT_ROOT% yum install privilege-xacml </pre>
   1 Replace /var/lib/tomcat[5|6]/webapps/gums/WEB-INF/lib/privilege-xacml-X.X.X.jar with the version you want to test <pre class="rootscreen">
%UCL_PROMPT_ROOT%cp /usr/share/java/privilege-xacml.jar  /var/lib/tomcat[5|6]/webapps/gums/WEB-INF/lib/privilege-xacml-X.X.X.jar</pre>
