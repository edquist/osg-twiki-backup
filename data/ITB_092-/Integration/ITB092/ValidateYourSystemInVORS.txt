---++ VORS Testing

VORS uses the site_verify.pl, located locally at $VDT_LOCATION/verify/ to run site level testing using the MIS VO. Failure (Red) indicates one of the following conditions. 
   * Failure to Authenticate (Check globus-gatekeeper.log at $VDT_LCOATION/globus/var/globus-gatekeeper.log for hints)
   * Failure to Run a Fork Job (Check globus-gatekeeper.log at $VDT_LCOATION/globus/var/globus-gatekeeper.log for hints)
   * Failure to Find a Job Manager (Check Job Manager specific log files)
   * Failure to Transfer files with GSIFTP correctly (Check $VDT_LCOATION/globus/var/gridftp.log)

---++ Validating VORS 

   * Check your system on the [[http://vors.grid.iu.edu/cgi-bin/index.cgi?grid=4&VO=0&region=0&res=0&dtype=0][VORS ITB]] page for:
      * does it exist?
      * does it pass?

      * review [[http://vors.grid.iu.edu/documentation/index.html][VORS Documentation]] for help
   * If these tests succeed, make %Y% entry in the validation table for your site.
