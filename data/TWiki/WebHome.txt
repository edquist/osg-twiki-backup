---+!! Welcome to the %WEB% Web

_The place to learn about TWiki features and perform TWiki system maintenance._

TWiki is a flexible, powerful, secure, and simple web-based collaboration platform.  Use TWiki to run a project development space, a document management system, a knowledge base or any other groupware tool on either an intranet or on the Internet.

---+!! List of all known TWiki Issues

This TWiki instance allows users to automatically login based on OIM registered X509 certificates. Some browsers are known to cause issues as described below.

---++ Firefox 4, 5, 6

No known issues

---++ Chrome 12, 13

No known issues

---++ IE 8, 9

No known issues

---++ Opera 11

No known issues

---++ Safari 5.1

On Mac OS 10.5 (Leopard), user can view, but not edit Twiki.

On Mac OS 10.6 (Snow Leopard), the browser crashes if user tries to access Twiki.

On Mac OS 10.7 (Lion), user can view, but not edit Twiki.

On all platforms(?), If you have X509 certificate stored on your machine, and if you click cancel button on the certificate selection dialog, Safari will not load any HTTPS webpages - including our twiki.
