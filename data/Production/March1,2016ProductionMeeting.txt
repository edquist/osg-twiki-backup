   * Vulnerabilities/Incidents
      * OpenSSL published security updates for DROWN (Decrypting RSA with Obsolete and Weakened eNcryption) attack
         * May allow a remote attacker to obtain sensitive information from servers that allow SSLv2 connections or that share a private key with any other server that allows SSLv2 connections. [[https://access.redhat.com/security/cve/cve-2016-0800][CVE-2016-0800]]
         * Disable SSLv2 in all the servers, if there is one server that supports SSLv2, the attack can be performed.
