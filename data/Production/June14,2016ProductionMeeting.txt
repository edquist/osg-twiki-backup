---++ Security
   * Vulnerabilities
      * OpenSSL vulnerability in CVE-2016-2108 rated as Important. Security updates for RedHat and Scientific Linux were made available. OSG announcement sent last week: https://ticket.opensciencegrid.org/30203
         * Security update for SL6 has been released: [[https://www.scientificlinux.org/sl-errata/slsa-20160996-1/][https://www.scientificlinux.org/sl-errata/slsa-20160996-1/]]
      * A buffer overflow vulnerability has been found for iperf3 utility (used in perfSONAR). An attacker can connect to an iperf3 server and by sending a malformed message on the control channel can corrupt server’s memory and perform a DoS attack. https://ticket.opensciencegrid.org/30219
   * Incidents
      * DoS attack reported at www.opensciencegrid.org website between June 11th and June 12th. Incident report at https://twiki.grid.iu.edu/bin/view/Security/OsgSec20160614
   * DCAFI project started transition phase. Intensity Frontiers started to being transitioned to CILogon Basic CA.
