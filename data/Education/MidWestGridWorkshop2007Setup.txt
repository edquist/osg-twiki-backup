%TOC{title="Outline:"}% 

---++ Setup CMS software environment
Source the =shrc= or =cshrc= file, according to your shell:
<verbatim style="background-color: lightblue;  text-align: left;padding: 10px; font-weight: 700; margin-left: 20px;margin-right: 20px;color: black;">
source /uscmst1/prod/sw/cms/shrc uaf
</verbatim>

---++ Create and Prepare user area
From your chosen user area, run this series of commands (for last command in list, use =-sh= or =-csh= as needed):
<verbatim style="background-color: lightblue;  text-align: left;padding: 10px; font-weight: 700; margin-left: 20px;margin-right: 20px;color: black;">
scramv1 p CMSSW CMSSW_1_2_0
cd CMSSW_1_2_0/src
eval `scramv1 runtime -sh`
</verbatim>
Note that if you redo the demo, you will not need to rerun the first command (=scramv1 ...=) again as it creates the structure; you will need to run the =eval ...= command from the right directory each time.

Now make and change to a "Demo" directory:
<verbatim style="background-color: lightblue;  text-align: left;padding: 10px; font-weight: 700; margin-left: 20px;margin-right: 20px;color: black;">
mkdir Demo
cd Demo
</verbatim>
