---++ Using host to find an IP address
<pre class="screen">
> <userinput>host www.amazon.com</userinput>
www.amazon.com has address 207.171.163.90
</pre>

---++ Using nslookup to find an IP address
<pre class="screen">
> <userinput>nslookup -sil www.amazon.com</userinput>
Server:         10.0.1.1
Address:        10.0.1.1#53
Non-authoritative answer:
Name:   www.amazon.com
Address: 207.171.166.102
</pre>

---++ Using dig to find more information about a domain
<pre class="screen">
> <userinput>dig www.amazon.com</userinput>
;; ANSWER SECTION:
www.amazon.com.         20      IN      A       207.171.166.102

;; Query time: 4 msec
;; SERVER: 10.0.1.1#53(10.0.1.1)
;; WHEN: Sat Jun 24 08:59:00 2006
;; MSG SIZE  rcvd: 48 
</pre>
