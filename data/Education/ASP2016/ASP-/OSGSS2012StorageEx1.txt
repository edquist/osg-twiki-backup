---++ Understanding blastp
The executable used for blast is ==blastp==.  It is located at ==%BLAST_DB_SUBMIT%==.  The goal of this exercise is to understand ==blastp== enough to be able to copy all files necessary for a single iteration into a directory.  ==blastp== is executed with: <pre class="screen">
%UCL_PROMPT% blastp -db %RED%database_file%ENDCOLOR% -query %RED%query_file%ENDCOLOR%
</pre>

I have provided an input query file in ==%BLAST_DB_SUBMIT%/query.fasta==

---+++ Input requirements
The input to blast is a database (file).  In the ==%BLAST_DB_SUBMIT%== directory, there is a few different databases.  For this exercise, attempt to only run the yeast db.


---+++ Output requirements
The output that we care about is sent to stdout.  Create a script that will put stdout into a file.
