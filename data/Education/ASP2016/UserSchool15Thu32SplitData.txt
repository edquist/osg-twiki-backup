---+ Example 3.2 Perform blast search for a different query file
The scripts and the workflow developed for BLAST search in the previous example is not limited to HLA.txt query file. It is easy to re-use the scripts with a minimal change for another query.   In this exercise, change the job scripts to download =http://stash.osgconnect.net/+dbala/Storage/MultiInputsMultiJobs-blastn/QueryFiles/BRAC1.txt = instead of the HLA file and use that for querying.
