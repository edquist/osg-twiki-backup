---++++ apply for an account and

 In order to do that, please follow *only* the [[http://www.ci.uchicago.edu/osgedu/schools/gridlab/intro.html][introductory part of the lab tutorial]] (denoted below as *Basics*), and allow 2-3 working days to receive all the approvals. %BLUE% *You can start the tutorial now. Good luck!* %ENDCOLOR% <br>

#SaveCert
---++ Saving your DOEGrids Certificate

   * Download the tool "pscp" from the PUTTY Web Site from: http://the.earth.li/~sgtatham/putty/latest/x86/pscp.exe into c:\pscp.exe
   * Insert your memory stick into your Lab computer (ideally into a USB slot ; )
   * Open a Command Prompt Window (Start > Accessories > etc )
   * cd to your memory stick
   * c:\pscp.exe !trainNN@gridlab1.ci.uchicago.edu:.globus/YOURNAME*.pem .
   * provide the trainNN password when prompted (and answer Y to the long prompt "The server's host key is not cached..." which comes before that)
