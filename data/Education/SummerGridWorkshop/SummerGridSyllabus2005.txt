---++ Summer Grid Workshop 2005 Syllabus

---+++Monday AM
	* *Lecture: Grid Intro and Fundamentals Review*
	* _Hands-on exercises: Explore basic sockets; ssh, scp and ftp; look at DNS_

---+++Monday PM
	* *Lecture: Grid Security and Basic Grid Access (GSI, jobs, file transfer, info)*
	* _FHands-on exercises: etch and install cert; create proxy; globusjobrun, gridftp_

---+++Tuesday AM
	* *Lecture: Grid Resources and Job Management (GRAM and Condor)*
	* _Hands-on exercises: CondorG, GRAM, RSL, Schedulers_

---+++Tuesday PM
	* *Lecture: Grid Data Management (!GridFTP, RLS, RFT, DRM)*
	* _Hands-on exercises: GridFTP (incl  3rd party), RLS, MCS_

---+++Wednesday AM
	* *Lecture: Grid Application Frameworks*
	* _Hands-on exercises: App packaging and install; running an app; data management; partitioning_

---+++Wednesday PM
	* *Lecture: Building, Monitoring, and Maintaining a Grid (OSG)*
	* _Hands-on exercises: Exploring and locating Grid resources; site independence_

---+++Thursday AM
	* *Lecture: Virtual Data Concepts*
	* _Hands-on exercises: Specifying workflow with VDL and interpreting/using provenance_

---+++Thursday PM
	* *Lecture: Grid Workflow and Resource Management*
	* _Hands-on exercises: Running workflow on the Grid; issues of large workflows_

---+++Friday AM
	* *Lecture: Web Services, the Resource Framework and the Grid (GT4)*
	* _Hands-on exercises: Create and run a basic WSRF service; addressing and notification_

-- Main.MichaelWilde - 08 Jun 2005
