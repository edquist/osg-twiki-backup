--+ !!Thursday Exercise 1.2: Do the OSG Connect Quickstart

For this exercise, you can also follow the online guide from the OSG Connect helpdesk that will acquaint you with submission on the OSG Connect submit server. Please use the submit host name user-training.osgconnect.net instead of login.osgconnect.net for the workshop. 

---++ Setup

   * Make sure you are logged into =user-training.osgconnect.net= (the OSG Connect submit server for this workshop).

---++ Follow the "OSG Connect Quickstart" tutorial

We will get the example files using the =tutorial= command.

<pre class="screen">
%UCL_PROMPT_SHORT% <strong>tutorial quickstart</strong>
%UCL_PROMPT_SHORT% <strong>cd tutorial-quickstart</strong>
%UCL_PROMPT_SHORT% <strong>ls -F </strong>
</pre>

You will see the following files:

<pre class="file">
Images/  osg-template-job.submit  short.sh*          tutorial02.submit
log/     README.md                tutorial01.submit  tutorial03.submit
</pre>

Take a look at the job description file =tutorial01.submit=.

<pre class="screen">
%UCL_PROMPT_SHORT% <strong>cat tutorial01.submit </strong>
</pre>

Let's submit the job file =tutorial01.submit= to HTCondor.

<pre class="screen">
%UCL_PROMPT_SHORT% <strong>condor_submit tutorial01.submit </strong>
</pre>
