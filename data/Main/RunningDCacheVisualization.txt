---++ Running the !dCache visualization in the ROC.

---+++ Installing
dCache Visualization requires [[http://developer.apple.com/technologies/tools/xcode.html][Xcode]] and [[http://www.libsdl.org/][SDL]].  Xcode is free, but requires sign up.  Xcode has an mac automated installer.  SDL is an open source library for cross platform rendering that is best built from source.

---++++ Install SDL (Need administrator access)
   1. Download the SDL source tarball from [[http://www.libsdl.org/download-1.2.php][SDL 1.2 Download]].  
   1. Open a terminal on the Mac, and change to the directory with the SDL.  If using firefox, it will be in =~/Downloads=.
   <pre class="screen">
cd ~/Downloads </pre>
   1. Untar the SDL tarball, replace version with the version number you downloaded.  If you are unsure on the version, type =SDL-= and begin pressing the tab button; the terminal should autocomplete the rest of the filename.
   <pre class="screen">
tar xzf SDL-1.2.%RED%version%ENDCOLOR%.tar.gz </pre>
   1. Move into the SDL directory
   <pre class="screen">
cd SDL-1.2.%RED%version%ENDCOLOR%.tar.gz </pre>
   1. Configure and make the SDL library.  You will be prompted for your password when it begins to install the library.
   <pre class="screen">
./configure && make && sudo make install </pre>


---++++ Download !dCache Visualization (Do not need administrator)
   1. Check out the ROC !dCache Visualization
   <pre class="screen">
svn co svn://t2.unl.edu/brian/HadoopViz/branches/ROC-Release-1.0.0 dCacheViz</pre>
   1. Configure and make the visualization.
   <pre class="screen">
./configure && make </pre>
