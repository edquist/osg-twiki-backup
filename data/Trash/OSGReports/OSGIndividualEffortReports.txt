---++ Individual Effort Reports

To add a name to the reporting structure edit this current page and add the name with no spaces (e.g. AnTemplate) to the list below in alphabetical order of the first name. Save this current page and select the "?" by the new name to create the new page for the individual report. Then copy the template from [[AnTemplate][An Template]] into this new page.
