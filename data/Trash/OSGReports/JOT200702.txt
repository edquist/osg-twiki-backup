SciDAC-2 Requirements on open source software: The VDT includes license information from each software product showing the open source nature of the contributions. The signed Statements of work include the following text about _source code_ developed by the project:

<blockquote>To comply with clauses defined in the <nop>SciDAC grant solicitation documents, any software produced as a result of the work done using <nop>SciDAC funds will be placed under open source licensing agreement.</blockquote>

---+++ Web Site
The OSG communications and administration staff have responsibility for the web presence for OSG: 
   * [[http://www.opensciencegrid.org][Web portal]] (http://www.opensciencegrid.org) for overview and communication, project overview, research plan, publications, presentations, interactions, progress reports; 
   * [[Main.WebHome][OSG Twiki]]-based collaborative documentation area where all activity and technical information;
   * [[http://osg-docdb.opensciencegrid.org/cgi-bin/DocumentDatabase/][Managed Document Repository]] for the Project and Consortiums reference documentation including security, agreements and policies. 
   * [[Main.MarciaTeckenbrock][Marcia Teckenbrock]] is the OSG document librarian (marcia AT fnal DOT gov).
   * [[Main.DavidSkinner][David Skinner]] from the <nop>SciDAC outreach center has agreed to attend the upcoming OSG All Hands meeting in March to discuss with the Communications group what information we can provide and how.
