<div style="font-family: Tahoma, Geneva, sans-serif;">
<div class="announcement-head" style="border: thin solid #000; background-color: #F90;">
  <h1 align="center">OSG 1.2.11 Update Notification</h1>
</div>
<br />

<div class="date" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; border: thin solid #000; background-color: #F93;">Date: <span class="date-text" style="font-size: 14px; font-weight: normal;">July 1, 2010</span></div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Affected Components</div>
<div>This update affects all OSG installations. </div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Summary</div>
<div> 
  <p>This update replaces Java 5 with Java 6 because Java 5 is past it's end of life and no longer has security updates. This affects all software in the OSG software stack that uses Java. In addition, we have updated to the latest version of Java 6, 1.6.0_20.</p></div>
<div>
  <p>This release updates several software components to new versions, see the complete list below.</p>

  <ul>
    <li>Java</li>
    <li>Java software has been altered to use Java 6 instead of Java 5</li>
    <li>osg-version</li>
  </ul>
</div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Issues Fixed</div>

<div>
  The default for Java software has been altered to use the Java 6 vm. In addition, Java 6 has been updated to the latest Sun Java release which has some bug and security fixes.</div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Update Instructions</div>
<div>Update instructions can be found on the OSG twiki under the OSG 1.2.11 <a href="https://twiki.grid.iu.edu/bin/view/Trash/ReleaseDocumentationOSG12UpdateInstructions">update instructions</a>.</div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Additional Information</div>
<div>The release notes for the VDT 2.0.0p18 release underlying this release can be found <a href="http://vdt.cs.wisc.edu/releases/2.0.0/release-p18.html">here</a>.</div>
