---++ Example user cert request

   * Setup the osg-client: <pre>
source /opt/client/setup.sh
</pre>
   * Then follow the instructions in CertScripts
   * Then retrieving: =cert-retrieve -serialnum 37709=

---++ Example host and http service certificate request

<pre>
cert-request -dryrun -agree -ou s -name "Steven C Timm" -host gc1-ce -email timm@fnal.gov -phone 630-840-8525 -reason "Colombia Grid School" -affiliation osg -vo osgedu -dir .
</pre>

   * remove the =-dryrun= and then the cert request will be submitted.
   * for httpd service add -service http
