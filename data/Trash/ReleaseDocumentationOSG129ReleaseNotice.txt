</div>
<br />

<div class="date" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; border: thin solid #000; background-color: #F93;">Date: <span class="date-text" style="font-size: 14px; font-weight: normal;">April 22, 2010</span></div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Affected Components</div>
<div>This update affects all OSG installations. </div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Summary</div>
<div> 
  <p>This is an update for several of the components in the OSG software stack. The changes are primarily to fix minor bugs or add some new functionality. Upgrading is not needed unless you are affected by one of these bugs or you need the new functionality provided in an update.<br />
  </p>
</div>

<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Components updated</div>
<div>
  <p>This release updates several software components to new versions, see the complete list below.</p>
  <ul>
    <li>Bestman 2.2.1.3.10</li>
    <li>Xrootd 20100315-1007</li>
    <li>XrootdFS 3.0rc4</li>

    <li>glExec 0.6.8.3-osg1</li>
    <li>PRIMA 0.8.7</li>
    <li>vdt-ca-manage</li>
    <li>CEMon 1.12.1-14</li>
    <li>Gratia 1.06.15i-1</li>
    <li>Pegasus 2.4.1</li>

    <li>OSG-RSV</li>
    <li>Java 1.6</li>
    <li>configure-osg</li>
    <li>osg-version</li>
    <li>wn-client (now includes gLite-FTS-Client and EDG-GridFTP-Client)</li>
  </ul>

  <p>The following packages are now available in the VDT cache</p>
  <ul>
    <li>gLite-FTS-Client</li>
    <li>EDG-GridFTP-Client</li> 
  </ul>
</div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Issues Fixed</div>
