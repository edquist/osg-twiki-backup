---+++ Network performance monitoring and tuning
   * Lead: Rich Carlson 
   * Topics: 
      * Introduction to perfSONAR monitoring framework 
      * Introduction to end-user diagnostic tools 
      * Hands-on troubleshooting tasks %u2013 using the tools to solve real problems 
   * Goals: 
      * Attendees will learn how to quickly respond to a user complaint about poor network performance. They will learn how to determine if the problem is caused by the network infrastructure, the host configuration, or the application behavior. Attendees will also learn how to quickly retrieve network performance data from remote locations, and how to find test points along the end-to-end path. 
   * References: 
   * Pre-workshop preparations: 
      * Attendees should bring an Intel based laptop to the workshop. A bootable CD with all the tools will be provided, this bootable disk will NOT touch the local HD or damage the laptop in any way. 
      * SLAC will provide a network switch and cabling to create a testbed LAN. This LAN should have access to the general Internet.

---+++ xrootd, Bestman II: OSG Storage
   * Lead: Tanya Levshina 
   * Topics: VDT packaging and configuration of Xrootd, xrootdFS,  Bestman-xrootd, Bestman-gateway  
   * Goals: Overview of packaging and installation procedure will be presented. Attendees  will be able to install and configure BeStMan "full-mode", BeStMan "gateway" mode and BeStMan-gateway/xrootd as well as VDT client that contains lbnl-srm, fermi-srm and lcg-utils commands.  Attendees will also be able to run validation tests suite.
   * References: 
   * Pre-workshop preparations: Attendees will need root access to one Linux node to install  BeStMan "full-mode", BeStMan "gateway" or at least two nodes to install  BeStMan-gateway/xrootd. Attendees will need a valid certificate for testing. 

---+++ dCache SE
   * Lead: Tanya Levshina (Steven Timm)
   * Topics:  dCache overview, opportunistic storage configuration, installation & configuration, dcache gratia probes
   * Goals: Overview of dCache, dCache packaging and installation will be presented. Attendees will be able to install and configure dCache and gratia probes  as well as VDT client that contains lbnl-srm, fermi-srm and lcg-utils commands.  Attendees will also be able to run validation tests suite.
   * References: 
   * Pre-workshop preparations: Attendees will need root access to at least one Linux node to install dCache and gratia probes. Attendees will need a valid certificate for testing.
