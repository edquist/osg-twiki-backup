---++ Starting the Live Image
Start the OSG Live Image by inserting the USB stick into your computer.  When starting your computer, you may need to enter the =Boot Selection= screen and select to boot from USB.

%ATTACHURL%/Screen_Shot_2012-02-20_at_12.32.22_AM.png

The !LiveUSB will boot and login to the desktop on it's own.

%ATTACHURL%/Screen_Shot_2012-02-27_at_4.00.06_PM.png


---++ Setting the Hostname for Condor-G jobs

In order to submit Condor-G jobs, you need to have a publically addressable hostname.  This means you need to have a public IP (as is the case in most grid software).  It might be that you will need to set the hostname of the machine.  To do so, first you need to find out if you have a proper hostname, and setting it.

<pre class="screen">
%UCL_PROMPT_ROOT% hostname %RED%&lt;hostname>%ENDCOLOR%
