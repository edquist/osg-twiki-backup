<div style="font-family: Tahoma, Geneva, sans-serif;">
<div class="announcement-head" style="border: thin solid #000; background-color: #F90;">
  <h1 align="center">OSG 1.2.10 Update Notification</h1>
</div>
<br />

<div class="date" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; border: thin solid #000; background-color: #F93;">Date: <span class="date-text" style="font-size: 14px; font-weight: normal;">June 10, 2010</span></div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Affected Components</div>
<div>This update affects all OSG installations. </div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Summary</div>
<div> 
  <p>This is an update for several of the components in the OSG software stack. The changes are primarily to provide xrootd changes requested by ATLAS and as well as to correct a few minor issues. This update is recommended mainly for admins interested in the new features present in the xrootd update.<br />
  </p>
</div>

<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Components updated</div>
<div>
  <p>This release updates several software components to new versions, see the complete list below.</p>
  <ul>
    <li>Xrootd</li>
    <li>Bestman, SRM-Client-LBNL, and SRM-Tester-LBNL 2.2.1.3.12</li>
    <li>MyProxy and GSIOpenSSH 5.1</li>

    <li>vdt-ca-manage</li>
    <li>Apache</li>
    <li>osg-version</li>
  </ul>
</div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Issues Fixed</div>
<div>
  This update fixes several minor bugs in software found in the OSG software stack and adds some new functionality to xrootd to support ATLAS and Trash/Tier3 requirements. Please see the <a href="http://vdt.cs.wisc.edu/releases/2.0.0/release-p17.html">VDT release notes</a> for more details.</div>

<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Update Instructions</div>
<div>Update instructions can be found on the OSG twiki under the OSG 1.2.10 <a href="https://twiki.grid.iu.edu/bin/view/Trash/ReleaseDocumentationOSG12UpdateInstructions">update instructions</a>.</div>
<br />
<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Additional Information</div>
<div>The release notes for the VDT 2.0.0p17 release underlying this release can be found <a href="http://vdt.cs.wisc.edu/releases/2.0.0/release-p17.html">here</a>.</div>
<br />
</div>
