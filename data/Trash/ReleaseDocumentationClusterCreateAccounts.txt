---++ Introduction
Here we describe creation of the needed accounts for the tutorial.  

---++ Account creation
Accounts must be created on the host =tg1-ce.yourdomain.org=. 

   * Account creation with =useradd= utility.
   * Create these accounts:  gsadmin (privileged user), gsuser (a normal interactive user), osgedu (a grid user account)
   * Creation of addon files (to append):
      * =group._addon.tg1= 
      * =passwd._addon.tg1=  
      * =shadow._addon.tg1= (shadow is write protected)
   * Push the files to each of the hosts in the cluster, appending them to the corresponding files as appropriate.
   * Check the files manually to see they were updated correctly.
