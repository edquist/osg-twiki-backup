---+++ Site Privileges for !ATLAS VO Support

*Minimum for Support*:
!ATLAS can make opportunistic use of site computing resources as long as !ATLAS VO members are mapped to some local account, and this account has write access to $OSG_APP area for the installation of !ATLAS software. 

*Recommended for Support*: 
These are the accounts and privileges for full site support for the !ATLAS !VO:

%EDITTABLE{ }%
|  *Account*  | *VOMS Attributes* | *Additional Info* |
| usatlas1 | /atlas/usatlas/Role=production | Account for managed production. |
| usatlas2 | /atlas/usatlas/Role=software | This account requires write access to $OSG_APP for the installation of !ATLAS software.  |
| usatlas3 |/atlas/usatlas | USATLAS members |
| usatlas4 | /atlas | non-US ATLAS members |

Definitions for these accounts and roles are part of the standard !OSG VO Package gums.template.

Information on how to update this package is available at https://twiki.grid.iu.edu/twiki/bin/view/Trash/DocumentationHowToUpdateVoLists.
