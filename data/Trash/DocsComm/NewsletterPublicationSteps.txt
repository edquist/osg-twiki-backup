---+++ Procedures to Publish OSG News

   * Newsletter is not in Xeno Media CMS, it must be reached via ftp (see Marcia for username and password)
      
   ftp address: forndebbil.tilted.net
   location of osgnews files:  /u/sites/opensciencegrid/htdocs/osgnews

   * I copy a previous newsletter in one of the yearly directories (not index.html at the top level) and edit in Dreamweaver.  
      
   * Newsletter must be previewed by EB prior to publication.  Name file "preview.html" and put in top osgnews
directory.  The corresponding http address is http://www.opensciencegrid.org/osgnews/preview.html
   * Test send newsletter before real send
   * Email newsletter, with a link to the newsletter included as well as the newsletter itself (Netscape Mail works best to preserve format.  
 	
   View Page Source and Copy Source 
 
In Netscape Mail, configure sender and reply-to address to be osgnews@opensciencegrid.org

Type beginning message, hit &#8220;Enter&#8221; twice and select:

   > Insert >> HTML

Paste in code and send.
