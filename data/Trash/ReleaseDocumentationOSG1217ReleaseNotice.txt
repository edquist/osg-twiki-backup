<div class="update-heading" style="font-family: Tahoma, Geneva, sans-serif; font-size: 14px; font-weight: bold; background-color: #369; border: thin solid #000; color: #CCC;">Summary</div>
<div> 
  <p>This release incorporates a variety of fixes and changes to various OSG software components. RSV has been changed to use a new <code>rsv-control</code> script that allows for more control when starting, stopping, and running RSV probes.  RSV now also has better support for a standalone RSV monitoring system. Bestman and Bestman2 have been updated to more recent version with several improvements and bug fixes. In addition, a new version of GIP has been included. The GIP update includes several job manager related bug fixes as well as support for WLCG site attributes..</p>
  <ul>
    <li>RSV has had a major overhaul to how it works. See the [[Trash/Trash/MonitoringInformation/RSV][RSV documentation]] for full details.</li>

    <li>Bestman and SRM-Client-LBNL have been updated to 2.2.1.3.18</li>
    <li>Bestman2, Bestman-Client, and SRM-Tester3-LBNL have been updated to 2.0.5</li>
    <li>SRM-Client-Fermi has been updated to 1.9.5-23</li>
    <li>GIP has been updated to 1.1.14 and now supports advertising of WLCG site information</li>
    <li>MyProxy has been updated to address a security issue</li>
    <li>Gratia has been updated and now supports HTPC accounting</li>

  </ul>
  <p>&nbsp;</p>
</div>
