---++ Resource and Service Validation (RSV) Configuration
%EDITTHIS%

The [[MonitoringInformation.RSV][OSG-RSV]] package is used for monitoring of the grid services on the CE.  It is installed with the OSG CE but not configured by default.

Unless you set =VDTSETUP_NO_CONDOR_CRON= to =1= in the [[PreparingComputeElement][CE pre-install document]], a minimal Condor package called condor-cron was installed to schedule !RSV probes.

RSV can be run on a standalone server if you would like to use if for monitoring multiple CEs or SEs (common on larger sites).  There are separate instructions for installing and configuring RSV as a [[https://twiki.grid.iu.edu/bin/view/MonitoringInformation/InstallAndConfigureRsvAdvanced#Install_standalone_RSV_package_f][<strong>standalone Package</strong>]].
