---++ Resource and Service Validation (RSV) Configuration
%EDITTHIS%

The [[MonitoringInformation.RSV][OSG-RSV]] package is used for monitoring of the grid services on the CE.  It is installed with the OSG CE but not configured by default.

Unless you set the =VDTSETUP_NO_CONDOR_CRON= environment variable to =y= while installing your CE (see: [[PreparingComputeElement][CE pre-install document]]), a minimal Condor package called Condor-Cron was installed to schedule !RSV probes.

RSV can be run on a standalone server if you would like to use it for monitoring multiple CEs or SEs.  There are separate instructions for installing and configuring RSV as a [[https://twiki.grid.iu.edu/bin/view/MonitoringInformation/InstallAndConfigureRsvAdvanced#Install_standalone_RSV_package_f][standalone Package]].
