---++Introduction
This is a tutorial to demonstrate how to install and setup a basic CE installation.  It will guide users through a basic CE installation step by step.  At the completion of this guide, you should have a simple CE available.

---++Requirements
You will need a server and root access to that server.  As long as the server is fairly modern (made in the last 7 years), you will be able complete this tutorial.  The installation will require about 5GB of space.  The server should have an full access to the internet and about 5GB of space available.  In addition, you should have a personal certificate and server, http, and rsv certificates as well.
