Generic Information Provider (GIP) Configuration

%EDITTHIS%
%BR%

---++ dCache Provider
If you need to advertise a dCache storage element in your Generic Information Provider, please note that you need to follow the instructions at [[InformationServices.Dcache.Gip][dCache Information Provider]]

---++ PBS Provider
PBS sites should refer to [[https://twiki.grid.iu.edu/twiki/bin/view/InformationServices/GipConf][GIP Configuration File]] to setup whitelisting and blacklisting for queues.

---++ More information
More information about the Generic Information Provider can be found at GenericInformationProvider.  Information found here include administrator overrides, removing GUMS status information being published through GIP, configuring GIP to not count VMs in "Owner" State in Condor Batch system, Squid monitoring, and adding site defined constraints.
