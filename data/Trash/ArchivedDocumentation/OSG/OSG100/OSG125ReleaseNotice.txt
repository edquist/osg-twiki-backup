<div class="announcement-head" style="background-color: #F90; border: thin solid #000;">
  <h1 align="center">OSG 1.2.5 Update Notification</h1>
</div>
<br />

<div class="date" style="font-size: 14px; font-weight: bold; font-family: Tahoma, Geneva, sans-serif; background-color: #F93; border: thin solid #000;">Date: <span class="date-text" style="font-size: 14px; font-weight: normal;">January 08, 2010</span></div>
<br />
<div class="update-heading" style="font-size: 14px; font-weight: bold; font-family: Tahoma, Geneva, sans-serif; background-color: #369; border: thin solid #000; color: #CCC;">Affected Components</div>
<div>This update affects all OSG based installs of Xrootd, XrootdFS, and BeStMan.</div>
<br />
<div class="update-heading" style="font-size: 14px; font-weight: bold; font-family: Tahoma, Geneva, sans-serif; background-color: #369; border: thin solid #000; color: #CCC;">Summary</div>
<div>
  <p>This update includes newer versions of several SE components that provide features requested by ATLAS.  Sites belonging to the ATLAS VO may need to apply this update to their SEs.</p>
  <p>We recommend this update if you are running  BeStMan, Xrootd, or XrootdFS and belong to the ATLAS VO. Other installations can consider this a low priority update.</p>

</div>
<br />
<div class="update-heading" style="font-size: 14px; font-weight: bold; font-family: Tahoma, Geneva, sans-serif; background-color: #369; border: thin solid #000; color: #CCC;">Components updated</div>
<div>
  <p>This release also updates several software components, see the complete list below.</p>
  <ul>
    <li>osg-version</li>
    <li>BeStMan</li>
    <li>Xrootd</li>

    <li>XrootdFS<br />
</li>
  </ul>
</div>
<div class="update-heading" style="font-size: 14px; font-weight: bold; font-family: Tahoma, Geneva, sans-serif; background-color: #369; border: thin solid #000; color: #CCC;">Issues Fixed</div>
<div>
  The components that have been updated may also include bugfixes. Please see the <a href="http://vdt.cs.wisc.edu/releases/2.0.0/release-p12.html">VDT release notes</a> for more details.</div>
<br />

<div class="update-heading" style="font-size: 14px; font-weight: bold; font-family: Tahoma, Geneva, sans-serif; background-color: #369; border: thin solid #000; color: #CCC;">Update Instructions</div>
<div>Update instructions can be found on the OSG twiki under the OSG 1.2.5 <a href="https://twiki.grid.iu.edu/bin/view/ReleaseDocumentation/OSG125UpdateInstructions">update instructions</a>.</div>
<br />
<div class="update-heading" style="font-size: 14px; font-weight: bold; font-family: Tahoma, Geneva, sans-serif; background-color: #369; border: thin solid #000; color: #CCC;">Additional Information</div>
<div>The release notes for the VDT 2.0.0p12 release underlying this release can be found <a href="http://vdt.cs.wisc.edu/releases/2.0.0/release-p12.html">here</a>.</div>
