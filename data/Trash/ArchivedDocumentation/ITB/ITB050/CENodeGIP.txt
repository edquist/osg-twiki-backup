---+++ Generic Information Providers (GIP)

%EDITTHIS%

---++++Introduction
The Open Science Grid (OSG) Generic Information Provider (OSG-GIP) has been developed to enable LCG-OSG interoperability, OSG resource selection service, [[Integration.GridCat][GridCat]], and to provide information to CEMon. The GIP is a grid information service that aggregates static and dynamic resource information for use with LDAP-based information systems. The GIP produces information based on the GLUE Schema.  For more information about OSG-GIP visit the [[http://grow.uiowa.edu/dokuwiki/doku.php/projects/gip/index][OSG-GIP webpage]].

---++++ Installation
The Generic Information Provider is installed with the standard OSG installation.  To activate it, you must complete the configuration section of this document.

---++++ Configuration
No extra work needed.

When you run configure-osg.sh it will execute configure-osg-gip.sh. This will create an gip-attributes.conf file that will record all the information needed to run configure_gip. The details of the information collected can be found in the [[http://osg.ivdgl.org/bin/view/Main/OSGConfigurationParameters#GIP][OSG Configuration Parameter]] twiki. The configure_gip script runs transparently without need for user input.

---++++ Post Install
None

---++++ Validation Webpage
A Validator for GIP can be found at [[http://grow.uiowa.edu/dokuwiki/doku.php/projects/gipvalidate/index][http://grow.uiowa.edu/dokuwiki/doku.php/projects/gipvalidate/index]].
