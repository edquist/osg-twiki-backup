---++ Introduction
Xrootd is a high performance network storage system widely used in high energy physics experiments such as ATLAS and ALICE. The underlying Xroot data transfer protocol provides highly efficient access to ROOT based data files.  This page provides instructions for using a Xrootd storage system consisting of one redirector node and one or more data server nodes.  

For installation instructions check XrootdInstallAlt.
