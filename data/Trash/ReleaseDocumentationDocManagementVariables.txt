---+++ Tracking documents for which you have some responsibility
There are a set of TWiki variables (command like) that will create tables on your home page (typically !Main.FirstLast) that will list documents for which you are the Owner, Reviewer, or Tester.  Variables also exist for Area owners of documentation.

---+++ How to create your own Home Page
   =The Twiki Variables= developed as part of the Content Management Project are available for every user to set up their Home Pages to the needs that suits them best. Discover the possibilities.

---++++ Preliminaries
   Please include the following HTML comment at the very end of every page you want to include into the "official" documentation of the OSG.%BR%

---++++ Predefined Document Tables
%NOTE% The queries below pick up the OWNER = !FirstLast from your home page. It will only work on your home page!  On other pages, you can define OWNER using<br><pre class="file">   * Local OWNER = FirstLast</pre>

---+++++ Include the Document Status Table
   To include a table showing the document status at the top right of the page use following Twiki variable right after the HTML comment:
<pre class="file">
%%NOP%DOC_STATUS_TABLE%
</pre>

%NOTE% The Document Status Table will be hidden whenever %%NOP%RELEASE_READY% = %%NOP%YES%.

---+++++ Include a Table of Documents that I own
   To include a table of documents that you own:
<pre class="file">
%%NOP%QUERY_OWNER_FORMAT%
%%NOP%TWISTY{%%NOP%TWISTY_OPTS%}%
%%NOP%SEARCH{%%NOP%QUERY_OWNER%}%
%%NOP%ENDTWISTY%
</pre>

---+++++ Include a Table of Comments for my Documents
   You can search all documents you own for comments that you or other users left:

<pre class="file">
%%NOP%QUERY_USER_COMMENTS_FORMAT%
%%NOP%TWISTY{%%NOP%TWISTY_OPTS_CLOSED%}%
%%NOP%SEARCH{%%NOP%QUERY_USER_COMMENTS%}%
%%NOP%ENDTWISTY%
</pre>

---+++++ Include a Table of Comments I made
   You can search all documents for comments that you made:

<pre class="file">
%%NOP%QUERY_OWNER_COMMENTS_FORMAT%
%%NOP%TWISTY{%%NOP%TWISTY_OPTS_CLOSED%}%
%%NOP%SEARCH{%%NOP%QUERY_OWNER_COMMENTS%}%
%%NOP%ENDTWISTY%
</pre>

---+++++ Include a Table of Tags I made
   You can search all documents you tagged. A tag is a special form of Comment that you can use on every page that includes %%NOP%Comments%. The format of the comment has to be 'TAG_NAME = TAG_VALUE', such as 'QUALITY = HIGH':

<pre class="file">
%%NOP%TAG_TABLE_FORMAT%
%%NOP%TWISTY{%%NOP%TWISTY_OPTS_CLOSED%}%
%%NOP%SEARCH{%%NOP%QUERY_OWNER_TAGS%}%
%%NOP%ENDTWISTY%
</pre>

---+++++ Include a Table of Documents that I am asked to Review
   Search all documents that have been assigned to you for review:

<pre class="file">
%%NOP%QUERY_REVIEWER_FORMAT%
%%NOP%TWISTY{%%NOP%TWISTY_OPTS_CLOSED%}%
%%NOP%SEARCH{%%NOP%QUERY_REVIEWER%}%
%%NOP%ENDTWISTY%
</pre>

%NOTE% A result will only be included once REVIEW_READY = %%NOP%YES% and only as long as RELEASE_READY = %%NOP%NO%

---+++++ Include a Table of Documents that I am asked to Test
   Search all documents that have been assigned to you for test:

<pre class="file">
%%NOP%QUERY_TESTER_FORMAT%
%%NOP%TWISTY{%%NOP%TWISTY_OPTS_CLOSED%}%
%%NOP%SEARCH{%%NOP%QUERY_TESTER%}%
%%NOP%ENDTWISTY%
</pre>

%NOTE% A result will only be included once TEST_READY = %%NOP%YES% and only as long as RELEASE_READY = %%NOP%NO%

---++++ Custom Searches
   Predefined searches are executed when the table is displayed. Custom searches allow you to define parameters and search on request.

---+++++ Create a Button that searches all Documents

<pre class="file">
%%NOP%FindAllFormAction%
%%NOP%TWISTY{%%NOP%TWISTY_OPTS%}%
%%NOP%FindAllFormat%
%%NOP%SEARCH{%%NOP%FindAllQuery%}%
%%NOP%ENDTWISTY%
</pre>

<div id="FindIt">
%FindAllFormAction%
%TWISTY{%TWISTY_OPTS%}%
%FindAllFormat%
%SEARCH{%FindAllQuery%}%
%ENDTWISTY%
</div>
