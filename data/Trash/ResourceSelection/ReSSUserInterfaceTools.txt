---+ !ReSS User Interface Tools

This page keeps a list of user tools to facilitate the interaction with the !ReSS system.

   * [[%ATTACHURL%/condor_status_ress][condor_status_ress]]: the following script prints out information from cluster classads in !ReSS in a summary table, much like condor_status. The script is meant to be extended by each VOs with the information most relevant to them.
<pre>
#!/bin/sh
echo
echo "GlueSiteName	GlueCEInfoContactString"
echo

condor_status -pool osg-ress-4  -format '%s\t' GlueSiteName -format '%s\n' GlueCEInfoContactString | uniq 
</pre>
