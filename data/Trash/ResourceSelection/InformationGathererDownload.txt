---+!! Information Gatherer Download Page

---++ Introduction

Following releases of Information Gatherer are available for download

---++ Releases
---+++ Version v1.0.12 (29 Jul 2011)
https://twiki.grid.iu.edu/twiki/pub/ResourceSelection/InformationGathererDownload/ress-1.0-12.noarch.rpm
---++++ Release Notes
   * Registered resources in OSG can now be fetched from MYOSG instead of OIM
---+++ Version v1.0.11 (04 Aug 2010)
https://twiki.grid.iu.edu/twiki/pub/ResourceSelection/InformationGathererDownload/ress-1.0-11.noarch.rpm
---++++ Release Notes
   * Greatly improved the performance of advertising the info to condor_collector. IG will advertise to multiple condor_collectors using one condor_advertise command.
   * BUG FIX: Fixed a bug which caused failure in stale classad file deletion under certain cirumstances
   * Add support for vo.allow and vo.deny. ig.properties should be configured with fnal.ress.ig.allow, fnal.ress.ig.deny with locations to these files. The vo.allow can be auto generated using get-registered-resources.sh with appropriate options. Each entry in these files is in format given below with support for wild characters.
     <pre> FQDNOfCENode #TAB# CommaSeparatedVOList</pre>
   * Changed the default value for fnal.ress.condor.classadfilelifetime in ig.properties to 15.
   * Changed the default value for fnal.ress.condor.expdate in ig.properties to 2020-01-01.


---+++ Version v1.0.10 (29 Oct 2009)
https://twiki.grid.iu.edu/twiki/pub/ResourceSelection/InformationGathererDownload/ress-1.0-10.noarch.rpm
---++++ Release Notes
   * Fixes a race condition while deployed in HA mode
   * IG does not add !StartdIPAddress to the generated condor class by default. 
   * Rpm will preserve the changes to the configuration files in /opt/ress-1.0/var/config
   * Added support for configuration parameter fnal.ress.condor.classadfilelifetime in ig.properties. Defaults to 30 min with minimum allowed to 10 min. At the beginning of every publication cycle, any classad files older than their lifetime are deleted.


---+++ Version v1.0.9 (20 Jan 2009)
https://twiki.grid.iu.edu/twiki/pub/ResourceSelection/InformationGathererDownload/ress-1.0-9.noarch.rpm
---++++ Release Notes
   * First version to be available for general download
