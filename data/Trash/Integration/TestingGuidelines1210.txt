---++ Specific Instructions for OSG 1.2.10

---+++ Test the new 'itb' keyword for installing certificates

When running vdt-ca-manage, install from the new ITB URL.  This should be done for all ITB sites.

Note: make sure to pick the appropriate location for your site.

<pre class="screen">
vdt-ca-manage setupca --url itb --location &lt;local|root|PATH&gt;
</pre>

   1. Are certificates installed successfully?
   1. Does any ITB-specific documentation need updating to remind admins to always install from this new URL?
