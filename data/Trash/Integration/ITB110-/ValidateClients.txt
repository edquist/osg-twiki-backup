---++ OSG Client

Below are a series of test you can do after installing the client software via the ClientInstallationGuide and sourcing the =setup.(c)sh= script.

---+++ *grid-proxy-init* and *voms-proxy-init*

Obtain a grid-proxy via =grid-proxy-init= (assuming your =usercert.pem= and =userkey.pem= files are in =$HOME/.globus/=)

<pre class=screen>
[user@clienthost test-submits]$ grid-proxy-init
Your identity: /DC=org/DC=doegrids/OU=People/CN=XXX XXXXX #####
Enter GRID pass phrase for this identity:
Creating proxy ....................................... Done
Your proxy is valid until: Wed Aug 22 04:57:38 2007
</pre>

Obtain a voms-proxy via =voms-proxy-init= (again assuming your =usercert.pem= and =userkey.pem= files are in =$HOME/.globus/=)

<pre class=screen>
[user@clienthost test-submits]$ voms-proxy-init -voms osg

Enter GRID pass phrase:
Your identity: /DC=org/DC=doegrids/OU=People/CN=XXX XXXXX #####

---+++ *Condor-G submission*

Create a test submit file
[neha@fgitb-gk test-submits]$ cat test.submit
universe = grid

globusscheduler = fermigrid1.fnal.gov/jobmanager-condor

executable = /bin/hostname

transfer_executable = false

log = cemon.log

output = cemon.out

error = cemon.err

stream_output = False

stream_error = False

queue

Then, submit it using command *condor_submit test.submit*

Check that job is submitted/runs successfully.
