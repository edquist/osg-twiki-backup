---++ Introduction to Condor Section

This section gives information about the Condor jobmanager.

---++ Options

The options given in bold type are mandatory.

| Option | Explanation |
| *enabled* | This indicates whether the Condor jobmanager is being used or not.  It should be set to True or False. |
| *home* | This should be set to be directory where fbs is installed |
| *condor_location* | This should be set to be directory where condor is installed.  If this is set to a blank variable or UNAVAILABLE, the configure-osg script will try to get this from the VDTSETUP_CONDOR_LOCATION environment variable. |
| *condor_config* | This should be set to be directory where the condor_config is located.   If this is set to a blank variable or UNAVAILABLE, the configure-osg script will try to get this from the VDTSETUP_CONDOR_CONFIG environment variable. |
| *job_contact* | This should be the contact string for the condor jobmanager (e.g. my.host.com/jobmanager-condor) |
| *util_contact* | This should be the contact string for the default jobmanager (e.g. my.host.com/jobmanager) |
| *wsgram* | This should be set to True or False depending on whether you want wsgram to be configured.  You will need to take additional steps to get wsgram running in addition to this. |
