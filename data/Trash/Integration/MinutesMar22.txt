---++ VDT 1.6.1c update
   * Details of the update are here: http://vdt.cs.wisc.edu/releases/1.6.1/
   * From Alain:
<pre>
There are two reasons to do the update:

1) If you are running PBS or LSF, you should update the Gratia Probe
   with the appropriate command:

   pacman -update Gratia-PBS-Probe
   or
   pacman -update Gratia-LSF-Probe

2) If you are on a multi-homed host and the GIP reports your host name
   incorrectly, you need to update the program that configures the GIP:

   pacman -update Configure-GIP

   After this update you can set GLOBUS_HOSTNAME in your environment
   to be the host name you wish to report. Then, rerun the configure_gip
   script to have it report your host name correctly.

If these problems don't affect you, there is no need to update.
</pre>
