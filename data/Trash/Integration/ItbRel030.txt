---++ Pacman Recommendation
The installation instructions are based on PacmanInfo.  For this release, we recommend *Pacman 3.13-1*.

---++ OSG Compute Element (CE) Description
The OSG CE installation is done with the Pacman command <verbatim>pacman -get iVDGL:osg-0.3.0.</verbatim> This delivers the following components:

	* Root package (pre-web services): <a href="http://hep.uchicago.edu/ivdgl/osg-0.3.0.pacman">osg-0.3.0.pacman</a> 
	* Root package (web services): <a href="http://hep.uchicago.edu/ivdgl/osg-0.3.0-ws.pacman">osg-0.3.0-ws.pacman</a> 
	* If you want both, you install 0.3.0, then 0.3.0-ws *in the same install directory*.
	* Depends on [[http://hep.uchicago.edu/ivdgl/osg-auto-0.3.0.pacman][osg-auto-0.3.0.pacman]] which sets some default answers to VDT configuration questions. It also distributes the OSG Release Notes. (Similarly, for the web services version.)
	* Depends on [[http://hep.uchicago.edu/ivdgl/VDT_137.pacman][VDT_137.pacman]]. Installed are the VDT packages:
	* Depends on [[http://hep.uchicago.edu/ivdgl/grid3-schema.pacman][grid3-schema.pacman]] which installs the Grid3 schema.
	* Dependent package: [[http://hep.uchicago.edu/ivdgl/osg-grid3-info-prov.pacman][osg-grid3-info-prov.pacman]] which installs the Grid3 information providers.
	* Dependent package: [[http://hep.uchicago.edu/ivdgl/vo-0.1.6.pacman][vo-0.1.6.pacman]] which installs the template files which assist resource administrators retrieve VO authorization information.
	* Dependent package: [[http://hep.uchicago.edu/ivdgl/MIS-CI.pacman][MIS-CI.pacman]] which installs the current Monitoring Core Infrastructure descibed below.
	* Dependent package: [[http://hep.uchicago.edu/ivdgl/srm-cp.pacman][srm-cp.pacman]] which installs the SRM client package.
	* Dependent package: [[http://hep.uchicago.edu/ivdgl/osg-gip.pacman][osg-gip.pacman]] which installs the OSG GenericInformationProviders package.
