---++  LBNL_VTB 

This site consists of 4 nodes that are reconfigured as needed for use in OSG Integration work as a Validation Test Bed. Each machine runs a hyperthreaded 3GHz Intel P4 CPU with 1GB RAM and has a pair of 250GB internal disks.

   *  osp1.lbl.gov is the CE node and master/submit host for local pbs/torque batch system
   *  osp2.lbl.gov is a worker node plus gums server for the cluster
   *  osp3.lbl.gov is a worker node plus stand-alone condor-g submit host
   *  osp4.lbl.gov is a worker node plus SRM/DRM tests system.

Cluster is administered by Main.JeffPorter

---++ LBNL_VTB_SE

   * Bestman SRM on osp4.lbl.gov originally used to test added HPSS functionality
      * approximately 200GB of disk w/  60GB of space reservation enabled
      * configured to use gums server on osp2.lbl.gov

Cluster is administered by Main.JeffPorter 

---++ NERSC_VTB00

This site is a small virtual cluster residing on a larger server machine.  Currently there is only 2 hosts running, each with 1cpu, 1GB RAM, ~20GB disks. This site is solely used for the basic installation tests in the VTB.

   *  osp-vtb00.nersc.gov is the CE node and master/submit host for local SGE batch system
   *  osp-vtb01.nersc.gov is a worker node and stand alone condor-g submit host

Cluster is administered by Main.JeffPorter
