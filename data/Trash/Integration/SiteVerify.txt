---++Introduction

<code>site_verify.pl</code> is a tool that performs basic validation tests on a grid computing resource.

---++Usage

By default, <code>site_verify.pl</code> will run a set of tests against the host you are running <code>site_verify.pl</code> from.  
You can also explicitly tell <code>site_verify.pl</code> what host to run against by using the <code>--host</code> option:

<code>
<pre>
$ perl site_verify.pl --host=myhost.domain.org
</pre>
</code>

To see the other options that can be delivered to <code>site_verify.pl</code>, use the <code>--help</code> option:

<code>
<pre>
$ perl site_verify.pl --help
</pre>
</code>
