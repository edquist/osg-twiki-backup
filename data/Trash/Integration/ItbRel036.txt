	* April 7:  ITB release candidate given to provisioning team %RED% (delayed till ~ April 21th) %ENDCOLOR%
	* March 20: Start of validations run for of deployed services and VO applications %GREEN% started March 23 %ENDCOLOR%
	* March 15: Date of ITB virtual integration meeting for OSG 0.3.6 %GREEN% done %ENDCOLOR%
	* March 8: All documentation on services and applications for installation, configuration and validation on ITB delivered. %GREEN% Done %ENDCOLOR%

	* Jan 31: ITB 0.3.6 Cache but not released until VDT 1.3.10 available %GREEN% (done) %ENDCOLOR%
