Describes the recommended hardware for the current OSG Integration Testbed. From [[Main.StevenTimm][StevenTimm's]] talk on [[http://osg-docdb.opensciencegrid.org/cgi-bin/RetrieveFile?docid=455][Lessons Learned & Best Practices]].

---++ Compute Element head node
   * At least 4GB of RAM. We have 6GB and we still swap.
   * At least 2 CPUs, as fast as you can get.
   * Enough disk to hold the VDT software and logs (~250GB).
   * For small­/medium size cluster, this machine can be an NFS server for home areas, too.

---++ For VirtualOrganizations/VOInfo Support
If you are supporting a VO, use a 2nd machine of with these specifications to be your VirtualOrganizations/VOInfoMS server.

---++ GUMS Support
If you are running GUMS, use a 3rd server with these specifications.
