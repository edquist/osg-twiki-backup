---++ Install standalone !RSV package, for example on a non-CE machine:

<p> To get latest !RSV v3.x (released in concurrence with OSG 1.2)
   * Ensure you have the pacman package manager, *if need be* get it:
 <pre class="screen"> cd /path/to/install/pacman
 wget http://physics.bu.edu/pacman/sample_cache/tarballs/pacman-3.28.tar.gz
 tar --no-same-owner -xzvf pacman-3.28.tar.gz
 cd pacman-3.28
 . setup.sh</pre>
   * Install !RSV package using pacman: Ensure you are =root= user, and then do the following
 <pre class="screen"> export NEW_RSV_LOCATION=/new/rsv/path; 
 mkdir -p $NEW_RSV_LOCATION
 cd $NEW_RSV_LOCATION
 pacman -get http://vdt.cs.wisc.edu/vdt_200_cache:OSG-RSV
 . $NEW_RSV_LOCATION/setup.sh </pre>

<p> NOTE: Version 2.4.x of RSV is deprecated, and not recommended for new installs.
