---++ Install standalone !RSV package, for example on a non-CE machine, using pacman

<p> Ensure you have the pacman package manager, *if need be* get it:

 <pre class="screen"> cd /path/to/install/pacman
 wget http://physics.bu.edu/pacman/sample_cache/tarballs/pacman-3.26.tar.gz
 tar --no-same-owner -xzvf pacman-3.26.tar.gz
 cd pacman-3.26
 . setup.sh</pre>

  <p> Installing !RSV package using pacman: Ensure you are =root= user, and then do the following
   <pre class="screen"> export NEW_RSV_LOCATION=/new/rsv/path; 
 mkdir -p $NEW_RSV_LOCATION
 cd $NEW_RSV_LOCATION
 pacman -get http://vdt.cs.wisc.edu/vdt_1101_cache:OSG-RSV
 . $NEW_RSV_LOCATION/setup.sh </pre>
