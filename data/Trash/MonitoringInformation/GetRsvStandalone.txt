---++ Install standalone !RSV package, for example on a non-CE machine:

<p> To get latest !RSV v3.x (released in concurrence with OSG 1.2)
   * If you do not have Pacman installed, [[ReleaseDocumentation/PacmanInstall][install it]].
   * Install the !RSV package using Pacman: As the =root= user do the following:
 <pre class="screen">
mkdir /opt/rsv
cd /opt/rsv
pacman -get http://vdt.cs.wisc.edu/vdt_200_cache:OSG-RSV
</pre>
