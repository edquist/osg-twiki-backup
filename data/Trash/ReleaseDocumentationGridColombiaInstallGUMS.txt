---++ Introduction
These are some modifications to a standard GUMS installation relevant for the workshop.
 
---++ Installation
   * Here are the standard instructions GUMSHandsOn
   * Change the install to use the workshop Squid server and Grid Colombia pacman cache: <pre>pacman -retry 3
pacman -allow trust-all-caches -http-proxy http://10.19.5.179:3128 -get http://osg-vtb.uchicago.edu/gco:gums</pre>

---++ Post-installation fix
There is a platform incompatibility between OSG and !CentOS that requires replacement of a script.
   * Down this file [[%ATTACHURL%/vdt-ca-manage.sh][vdt-ca-manage.sh]]: Replacment vdt-ca-manage.sh script
   * cd to your GUMS install area where VDT keeps its scripts, eg: =/opt/gums/vdt/bin= and replace the script.
