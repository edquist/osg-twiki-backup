---++ Install Instructions
   1 Install rsv-vo-gwms package <verbatim>yum install rsv-vo-gwms --enablerepo=osg</verbatim>
   1 Set up a proxy file readable by rsv user and and with vo VOMS extension. It needs to be renewed outside of rsv (possible via a cron job). An example running in the rsv crontab would look like this:<verbatim>* */12 * * * /var/rsv/.globus/refresh_proxy.sh </dev/null >/dev/null 2> /dev/null</verbatim> And the refresh_proxy script like:<verbatim>#!/bin/bash                                                                                                                                                   
voms-proxy-init -cert /etc/grid-security/rsv/rsvcert.pem -key /etc/grid-security/rsv/rsvkey.pem -voms osg -out /etc/grid-security/rsv/mis.proxy -hours 72</verbatim>
   1 Modify file <verbatim>/etc/osg/config.d/30-rsv.ini</verbatim>  and set <verbatim>enable_gratia = False</verbatim> and the rsvcert and rsvkey paths <verbatim>service_cert</verbatim>, <verbatim>service_key</verbatim>, and <verbatim>user_proxy = etc/grid-security/rsv/mis.proxy</verbatim> 
   1 Run <verbatim> osg-configure -v</verbatim> and then <verbatim> osg-configure -c</verbatim>
   1 Modify the file <verbatim>/etc/rsv/rsv.conf</verbatim> and set <verbatim>proxy-file </verbatim> to the actual path of the proxy. (For example /etc/grid-security/rsv/mis.proxy)
   1 Start httpd, condor-cron and rsv services. 
   1 Run <verbatim> rsv-control --enable org.osg.local-gfactory-querying-local  --host fermicloud053.fnal.gov </verbatim> where host can be localhost or the actual name of the node where rsv-vo-gwms is installed.
   1 Run <verbatim>rsv-control --on org.osg.local-gfactory-querying-local  --host fermicloud053.fnal.gov </verbatim> where host can be localhost or the actual name of the node where rsv-vo-gwms is installed
