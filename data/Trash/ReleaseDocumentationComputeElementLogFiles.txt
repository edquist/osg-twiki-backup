---+ About this Document

This document intended audience are sys admins managing Compute Elements. The document lists locations and briefly describes contents of log files generated by processes that constitue an OSG Compute Element.

Below is a table for locating log and error files for common services located in the OSG CE installation (all are relative to your =$VDT_LOCATION=).
If this table gets out of date and you cannot find a log file you are looking for, a convenient way to search for log files is:
<verbatim>
$ find $VDT_LOCATION -name "*log" -print
</verbatim>
At the same time, if you notice differences and/or missing logs - please [[https://ticket.grid.iu.edu/goc][open a GOC ticket]]
