---++ Proposal to add x509 authentication to TWiki
   * A change that would allow users to authenticate with their existing certs.

---++ Proposal to truncate database
   * The database behind  the MyOSG "RSV Status History" display is getting very large (58 GB). Extrapolation indicates we will run out of disk space in ~1 year. It is proposed that entries older than one year be removed from the working copy. A full backup will be maintained. Queries needing data older than one year could be handled on a case by case basis.

---++ BDII stress test
   * At the suggestion of Burt H., Brian B., Igor S. in coordination with the GOC are proposing to stress test the BDII-itb. A script has been written that approximates a real use case request. It is proposed that this script will be run against the BDII-itb from many geographically distributed sites.
      * BDII-itb currently running a recent version (5.1.7)

      * munin tests installed to monitor/plot machine status
