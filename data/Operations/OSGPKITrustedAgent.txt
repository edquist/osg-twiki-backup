---++ Becoming an OSG PKI Trusted Agent (Registration Authority Agent or Grid Admin)

   * Review training materials (TBD) and [[https://twiki.grid.iu.edu/twiki/pub/Operations/OSGPKIAgreements/DigiCert-OSG_Agent_Agreement.pdf][Trusted Agent Agreement]]
      * RA Agents are authoritative for issuing user certificate for a VO or site.
      * Grid Admins are authoritative for issuing host and service certificates for a site (or sites).
      * Requesting RA Agents and Grid Admins must be registered with OIM.

---+++ To Request to become a Registration Authority Agent for a VO or site
   * In a web browser in which your certificate is installed, please visit the webpage for the [[https://oim.grid.iu.edu/oim/vo][VO]] you wish to become a RA for.
   * Please make use you have coordinated with any existing RA Agents for the VO.
   * Select the "Request for RA Enrollment" button on the page
   * You will be presented with a form letter, please fill in your name and VO/site name as directed, and Submit the request.
   * You will be contacted by the OSG PKI Operations staff when your request has been approved.

---+++ To Request to become a Grid Admin
   * In a web browser in which your OIM-registered certificate is installed, please visit [[https://oim.grid.iu.edu/oim/gridadmin][Grid Admins]] web page
   * Please make use you have coordinated with any existing Grid Admins for the domain(s) you wish to be a Grid Admin for.
   * Select the "Request for Grid Admin Enrollment" button on the page
   * You will be presented with a form letter, please fill in your name and the domain name(s) as directed, and Submit the request.
   * You will be contacted by the OSG PKI Operations staff when your request has been approved.

---++ Annual Maintenance of Role

   * Each year, RAs and GAs will be required to re-affirm their commitment to their responsibilities, and refresh their training. ([[https://jira.opensciencegrid.org/browse/OSGPKI-178][Implementation of this refresh is pending.]])
