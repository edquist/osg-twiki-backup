---+ Getting Resource Specific Data from !MyOSG

---++ Introduction

!MyOSG provides views of OSG data consolidated onto a single location. This document will explain how to get OSG Resource data. Similar options are available for Support Center and Virtual Organization information, but will not be covered in this document.

---++ Visualization Options

!MyOSG comes with a display wizard, which can be found [[http://myosg.grid.iu.edu/wizard?all_resources=on&gridtype=on&gridtype_1=on&summary_attrs_showservice=on&summary_attrs_showrsvstatus=on&summary_attrs_showfqdn=on&gip_status_attrs_showfqdn=on&gip_status_attrs_showtestresults=on][here]]. The following instructions will greet you at this link.

---+++ !MyOSG Resource Main Page

Please follow these 3 easy steps.

1. Start by selecting "Information to display" on the right.

2. Select resources that you'd liked to see the requested information.

3. Optionally, you can filter the list of resource by applying various filters.

* Click Update button when you change your query
