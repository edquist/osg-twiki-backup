---++ GGUS updates
<pre>

In parallel to the actions listed in savannah items
https://savannah.cern.ch/support/?110960 and
https://savannah.cern.ch/support/?113831
we are going to upgrade the GGUS Remedy system (ARS) from version 7.1 to version 7.5.
Besides the changes mentioned in the savannah items above the following things will
definitely change:
- the URL of the web service interface
- the OpenSSL version

The upgrade for the GGUS   t e s t   instance is scheduled for the 14th calendar week.
The wsdl URL of the SOAP API will change from the current URL
https://iwrgustrain.fzk.de/arsys/WSDL/public/iwrgustrain/Grid_HelpDesk
to the new URL
https://train-ars.ggus.eu/arsys/WSDL/public/train-ars/Grid_HelpDesk
OpenSSL will be upgraded to version 1.0.0

Please be prepared to change the wsdl URL for GGUS   t e s t   instance in calendar week 14.
Problems in https communcation may also occur due to the OpenSSL upgrade to version 1.0.0.

The upgrade of the GGUS   p r o d u c t I o n   instance is scheduled for the April
release. A date for the April release is not fixed yet, but it will be by end of April or
early May.
Relevant changes will be broadcasted via savannah ticket
https://savannah.cern.ch/support/index.php?119899 and this mailing list in time.

If you have any question or comments please let us know.
</pre>
