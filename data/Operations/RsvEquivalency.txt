---+ RSV WLCG SAM Equivalences

---++ Computing Element (CE):

   * <code>CE-sft-caver</code> is equivalent to <code>org.osg.certificates.cacert-expiry</code>
   * <code>CE-sft-csh</code> is equivalent to <code>org.osg.general.osg-directories-CE-permissions</code> 
   * <code>CE-sft-lcg-rm</code> is equivalent to <code>org.osg.globus.gridftp-simple</code> 
   * <code>CE-sft-job</code> is included in <code>org.osg.general.osg-version</code> 
   * <code>CE-sft-softver</code> is equivalent to <code>org.osg.general.osg-version</code> [Should this be org.osg.general.vo-supported ? -AG]
   * <code>CE-host-cert-valid</code> This one does not apply for OSG (no way to check)
   * <code>CE-sft-brokerinfo</code> This one does not apply for OSG (no broker)
   * This is not available in EGEE  in OSG <code>org.osg.general.ping-host</code>
   * This is not available in EGEE; in OSG <code>org.osg.globus.gram-authentication</code>

---++ Storage Element (SE):

   * These do not apply to OSG as they only have SRMs
   * <code>SE-lcg-cr</code>
   * <code>SE-lcg-cp</code>
   * <code>SE-lcg-del</code>

---++ SRM (SRM):

   * <code>SRM-put</code> is equivalent to <code>org.osg.srm.srmcp-readwrite</code>
   * <code>SRM-get</code> is equivalent to <code>org.osg.srm.srmcp-readwrite</code>
   * <code>SRM-advisory-delete</code> is equivalent to  <code>org.osg.srm.srmcp-readwrite</code>
   * This one does not apply to EGEE ; in OSG <code>org.osg.srm.srmping</code>


---++ Site BDII (sBDII):

   * Does not apply as OSG only uses a global BDII. A future release of RSV will include Global BDII tests.
   * <code>sBDII-sanity</code>
   * <code>sBDII-performance</code>
